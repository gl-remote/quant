## 一、缺失项目元素

### 1.1 工程基础设施

| 元素 | 现状 | 优先级 |
|------|------|--------|
| 测试框架与用例 | 无 `tests/` 目录，零测试覆盖 | 紧急 |
| CI/CD 流水线 | 无 `.github/workflows` | 高 |
| 代码质量工具配置 | pytest/flake8/pylint 无对应配置文件 | 高 |
| 项目打包配置 | 无 `pyproject.toml` 或 `setup.py` | 中 |
| 版本锁定 | requirements.txt 使用 `>=` 宽松约束 | 高 |
| 许可证文件 | 无 `LICENSE` | 中 |
| 编辑器配置 | 无 `.editorconfig` | 低 |
| Docker 支持 | 无 `Dockerfile` | 低 |

### 1.2 文档缺失

| 元素 | 说明 |
|------|------|
| 变更日志 (CHANGELOG.md) | 无版本变更记录 |
| 贡献指南 (CONTRIBUTING.md) | 无代码贡献规范 |
| 策略开发指南 | 仅 FAQ 中简要提及 |

### 1.3 功能缺失

| 功能 | 说明 |
|------|------|
| 多品种并发回测 | 仅支持单品种 |
| 回测结果可视化 | 无本地 matplotlib 图表 |
| 参数优化/网格搜索 | 无内置超参搜索 |
| 实时风控监控 | 无回撤熔断、日亏损限额 |
| 多策略支持 | 仅均线交叉一个策略 |
| 通知/告警 | 无异常通知机制 |
| 多数据源 | 仅支持天勤 |
