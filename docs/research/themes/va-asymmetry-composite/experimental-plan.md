## §0 定位与评估框架

### §0.0 本轮定位：从"找信号"转向"能用→调优"

- **背景**：`docs/archive/strategy-research/2026-07-10-va-asymmetry-composite` 已完成首轮基准——
  B0 达 **Sharpe 2.70 / 年化 15.10% / MaxDD −2.40%**，6 tier 骨架在该轮已验证"能用"，且 `W/VW/S2` 经 §11 审计否决。

> ⚠ **B0 锚点口径澄清（2026-07-11，P0.1 干净重做后·推翻旧论断）**：此前一度认为旧 stage4 逐事件行滚 `ROLLING_EVENTS=100` 是"虚高结果的 bug"，并据**混淆版** `timeline_calA`(11.50%/2.23) 把 B0 改为更弱数字、宣称"归档 2.70 偏高"。**经干净最小修复（只隔离 skew 去重、对齐旧管线其余）三路 A/B 验证，该前提不成立**：
> - 冻结管线 B0（同引擎复现，新·可交易日口径=只用 skew 拿到值的交易日）= **年化 26.12% / 净夏普 3.62 / MaxDD −2.51%**（312 笔，可交易日 590）；旧日历口径为 15.68%/2.72，与归档 B0(15.10%/2.70/−2.40%) 一致 → 冻结管线即策略真相。
> - 唯一识别的"缺陷"（skew 去重）实测为**退化而非修复**：年化 15.68%→**13.14%**、夏普 2.72→**2.40**、MaxDD −2.51%→**−4.24%**、交易 312→216（**−31%**）。机制：去重后窗口=100 需 100 个不重复日观测，短历史合约 skew 秩整体变 NaN 出局，非 spec §1.3.0 假设的"≤0.05 jitter"。
> - **结论**：撤回"2.70 偏高 / B0=2.23"论断，**B0 锚点恢复为 ~15.7%/2.72**（同引擎口径）；混淆版 `timeline_calA.parquet` 废弃。下游 P1/P5 据冻结管线重审（见对应章节修正）。详见 `project_data/ai_tmp/p0_calib/ab_compare.md`。
- **本轮姿态**：不是"从头找信号"，而是**在已验证可用的骨架上做增量调优**。因此：
  - 原"分类器骨架 / 6 tier edge 重验"**不再作为最高杠杆强 gatekeeper**，降级为 **P0 内的新口径轻量确认**（见 P0.4）。
  - 真正的调优杠杆按归档实证结论排序：`Cap`（名义暴露）> `H_vol(tier)`（持仓时长分化，P2 后升格为真实候选轴）> 其余局部塑形/风控。
- **关键免责（口径变更，2026-07-11 修正）**：旧 stage4 在**未去重**的逐事件行上滚 `ROLLING_EVENTS=100`，spec §1.3.0 建议改为 `drop_duplicates(event_date)` 后按天滚动。但干净最小修复的 A/B 显示：**该去重改动在 B0 上实测为退化**（年化 15.68%→13.14%、夏普 2.72→2.40、交易 −31%，见上方 B0 锚点澄清），机制是窗口=100 在不重复日不足时覆盖崩塌。**故 spec §1.3.0 的"去重更正确"假设在本数据集上未被验证、反而有害**；冻结管线（未去重 skew 秩）保留为有效基线，"tier 能用"直接继承（P0.4 仍做轻量确认，但不再以"去重必要"为前提）。

- **核心动机（归档实证）**：年化瓶颈在名义暴露压缩——日均 653% 被 `Cap=100%` 砍掉 **85% 交易**，是卡在 15% 未达 18% 的高概率根因。本 Phase 直接对准该瓶颈放大有效信号的暴露。
- **判定**：相对 B0 净夏普增量 ≥ 0.2 且 P≥0.95 方采用；仅放大波动不增夏普者不采用。
- **失败处理（§3）**：若放大暴露后夏普不增反降 → 信号可能本就边际（呼应"朴素等权 1/N 谜题"，spec §11 已否决加权）→ 回到 P0.4 复核信号强度，而非继续下游。

> **【结论（2026-07-11，冻结管线，同引擎 `va_composite_p1_cap`；见 `p1_cap/summary.md`）】** 此前基于混淆版 `timeline_calA`(11.50/2.23) 的"维持 B0/夏普随 Cap 单调降"已被干净 A/B 推翻，现据冻结管线重审：
> - **主指标（dedup=8h · 新·可交易日口径=只用 skew 拿到值）**：B0(Cap=1.0) **26.12% / 3.62 / −2.51%**（312 笔，压仓 92 天）→ Cap=1.2 28.02%/3.53/−3.23% → 2.0 31.93%/3.39/−4.26% → 4.0 35.42%/3.50/−4.95% → 5.0 **35.82% / 3.50 / −5.09%**（压仓 3 天）。
> - **配对增量（vs B0@1.0，§0.1 门）**：Cap=1.2 ΔSh+3.46/P1.000、2.0 +3.47/0.999、4.0 +3.91/1.000、5.0 +3.79/1.000 → **均过门**。说明放大暴露注入的是**真实正均值**交易（杠杆非纯噪声）→ Cap 确为有效轴。
> - **诚实解读（领先于程序正确性·契合"朴素 1/N 谜题"先验）**：随 Cap 上升，**年化 26.12%→35.82%（+9.7pp）但净夏普 3.62→3.50（基本持平、略降）**——放大暴露以"放大回撤(−2.51%→−5.09%)、牺牲边际夏普"换更高绝对收益，并非"免费午餐"。这正印证 spec §11 已否决加权/杠杆后"优化多半跑不赢 B0 夏普"的先验：**Cap 提升是收益/回撤的预算扩张，不是夏普优化**。
> - **采用建议（2026-07-11 修订·综合夏普+年化选定）**：**默认锁定 Cap=4.0**——综合"夏普≈杠杆不变（3.62→3.50 基本持平）+ 年化≈4×（26.12%→35.42%）"选定；代价是回撤≈4×（−2.51%→−4.95%）与尾部风险放大，须明确接受。1.0 档仅"夏普绝对数最高（3.62）/年化最低"，已让位于 4.0；5.0 档（35.82%/3.50/−5.09%）边际收益极小（年化仅 +0.4pp）且回撤更深，故止于 4.0。配对门 1.2~5.0 全过（ΔSh≥+3.4/P≥0.999），Cap 确为有效轴。撤回此前 calA 版"维持 B0 因各 Cap 夏普都更低"——该判断建立在更差的伪基线上；真实冻结管线下各 Cap 夏普本就与 B0 持平。

## §3 失败回溯协议（归档实证 + 领域先验）

> 依据研究备忘：当某层候选**全部 0/N 未过门**时，首要假设是"这批旋钮/轴本身选错了/是弱轴"，
> **而非"基线神奇最优"**；应主动回溯上游找真正杠杆，而非继续拧下游。
> 本轮额外锚定归档实证：组合加权已 exhausted，真实杠杆在**名义上限 + 事件层（持仓时长 `H_vol` / 事件选择治理，而非日内择时 `entry_mode` 已证伪）**。

- **P0.4 失败（新口径下 ≥2/6 tier 反转负）** → 最强信号：新管线使分类架构系统性失效。先质疑 `drop_duplicates` 滚动口径是否引入偏差、或归档"tier 能用"本就依赖旧口径缺陷 → **回到 P0.1 复核管线语义**，不进入 P1~P9。
- **P1 失败（Cap 放大暴露不增夏普）** → 信号边际：呼应"朴素等权 1/N 谜题"（spec §11 已否决加权）。回到 P0.4 复核信号强度；**不要**假设"Cap 没调对"。
- **P2（7 mode 全 0/N，已执行并确认）** → 择时为零/负轴，已审计进 spec §11；真实"事件层"杠杆转 `H_vol(tier)`（P6 主杠杆②）与 `sym(tier)` 裁剪（P9）。不再回溯 P0.4 复核 entry_mode（其结论已落定，重测不产生新信息，除非有强烈反转证据单独立项）。
- **P3~P7 局部轴普遍 0/N** → 符合"局部塑形增量本应小"的先验；确认 B0 已接近该维度最优，不强行采用。若**同时**多个局部轴均显负向拖累 → 警惕过拟合，回到 P0.4/P4 查信号稳健性。
- **全局 0/N（多层皆弱）** → 不继续加轴；重新评估"分类架构是否整体选错轴"，考虑回到 tier 定义/事件选择层级（spec §1.2 元逻辑）重做，而非在现有 6 类框架内继续调参。
- **已否决轴（W/VW/S2，spec §11）不纳入回溯**：其失败已被审计记录，重测不产生新信息；若新证据强烈反转，单独立项而非并入本计划。
