## 3. 共同教训

### 强度识别 ≠ 方向识别

- **方向 alpha**（通道 A）只需二分类（P(win) > 1/(1+R)），可借力结构性不对称
- **强度 alpha**（通道 B）需连续回归精度（se ≤ 0.05），无结构性锚点
- 8 个跨类型因子（时序 / 横截面 / 成交量 / 制度切换）Gate 3 秩相关 r ∈ [-0.126, 0.061]
- **结论**：单独推算市场信号强度目前看并不比推断方向容易

### GBM/Markov 无记忆假设的经验验证

- 过去价量统计量不包含未来漂移强度的预测信息
- 连"制度切换"型因子（日内时段效应）也几乎不存在（时段间均值差仅 1.5%）
- 唯一未穷尽的路径：F9 事件驱动外源信号（需外部数据源）
