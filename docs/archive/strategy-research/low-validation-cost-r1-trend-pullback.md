## 1. 实验目标

本轮不重复验证布林带收缩后边界，改测 `low-validation-cost-r1` 的另一个候选区间：**趋势背景下回踩区域**。

核心问题：

```text
趋势背景下的回踩区，是否能提供客观、近距离的严格失败边界，
并允许用有限止损放宽购买足够胜率？
```

本轮不是寻找最优参数，也不做大规模 Optuna 搜索，只验证最小策略结构是否能产生可评价样本。

## 2. 区间定义

| 项目 | 本轮定义 |
| --- | --- |
| 实验版本 | `low-validation-cost-r1` |
| 策略实现 | `lvc_pullback` |
| 候选价格区间 | `15m` 趋势背景下，`5m` 回踩到趋势均线附近后重新接受 |
| 结构来源 | 趋势回踩 / 均线回踩 |
| 传统解释 | 趋势回踩后重新转强 / 转弱，顺趋势延续 |
| 结构塑形解释 | 回踩低点 / 高点天然形成近距离严格失败边界，可低成本验证趋势延续假设 |
| 多头方向假设 | `15m` 快均线在慢均线上方，`5m` 回踩后重新上破前一根高点并收回趋势均线上方 |
| 空头方向假设 | `15m` 快均线在慢均线下方，`5m` 反弹后重新跌破前一根低点并收回趋势均线下方 |
| 严格失败边界 | 多头为本轮回踩低点；空头为本轮反弹高点 |
| 严格失败距离 | 入场价到严格失败边界的距离 |
| 实际止损边界 | 严格失败距离按固定 `stop_widen_multiplier` 有限放宽 |
| 目标止盈 | 按实际止损距离的 `target_r_multiple` 小目标 |
| 时间退出 | 超过 `max_hold_bars` 未触达止盈或实际止损则退出 |

## 8. 观察结论

本轮已经从“样本生成失败”修正为：

```text
趋势回踩候选区间能产生足够样本；
0 交易来自诊断日志字段类型导致的回测异常，不是区间没有信号。
```

但固定参数回测和止损放宽对照结果明显不通过：

- 三个合约总计 `223` 条交易记录，样本数量足够继续诊断；
- `1.5` 倍放宽止损只把平均胜率从 `38.96%` 提高到 `40.28%`，提升很小；
- 放宽后平均单笔期望从 `-541.82` 恶化到 `-654.66`，合计收益率从 `-129.57%` 恶化到 `-151.08%`；
- `DCE.m2601` 放宽后胜率提升明显，但平均亏损扩大到 `4650.00`，收益仍更差；
- `DCE.m2603` 与 `DCE.m2605` 放宽后胜率下降，说明放宽没有稳定购买胜率；
- 当前问题优先不在止损倍数，而在“重新接受”后的方向质量和止盈/时间退出结构。

当前结论：

```text
lvc_pullback 可以生成趋势回踩候选样本，但默认重新接受定义没有形成正向优势；
1.5 倍止损放宽没有改善成本后期望，不能继续沿扩大止损方向优化。
```

## 10. 观察结论

本轮已经从“样本生成失败”修正为：

```text
趋势回踩候选区间能产生足够样本；
0 交易来自诊断日志字段类型导致的回测异常，不是区间没有信号。
```

但固定参数回测、止损放宽对照和 MAE/MFE 诊断共同说明：

- 当前入场确认质量不足，失败交易的 MFE 很小、MAE 很大；
- 成功交易通常入场后回撤很小，说明“好样本”存在，但当前条件无法有效筛出它们；
- 放宽止损会让平均 MFE 增加，但 MAE 增幅更大，尤其亏损交易平均 MAE 从 `8.76` 扩大到 `11.82`；
- 时间退出不是主要错过大行情的问题，`time_exit` 的 MFE/MAE 不支持继续延长持仓；
- 当前问题优先不在止损倍数，而在“重新接受”定义过弱，允许太多假重新接受进入。

当前结论：

```text
lvc_pullback 可以继续研究，但方向应从“止损放宽”切换到“重新接受质量过滤”；
下一轮优先测试更强重新接受定义，而不是继续扩大止损或延长时间退出。
```
