## 1. 补充动机

原结构型 alpha 实验主要回答：

```text
某个共识边界 / 接受拒绝 / 假突破 / 重新接受结构，
作为独立入口后，能否直接形成成本后正期望？
```

随机对照补充计划回答更基础的问题：

```text
这些结构入口是否至少比随机开仓更有信息量？
```

如果结构入口与匹配随机入口无差异，则结构应暂停；如果结构显著优于随机但仍未转正，则说明结构仍有研究价值，后续应转向成本空间、账户风险预算、退出结构、品种适配和尾部风险补证。

## 2. 对照对象

计划覆盖的结构路线：

| 结构路线 | 阶段处理 |
|----------|----------|
| Initial Balance 假突破重新接受 | 历史报告存在，策略实现缺失，无法同口径复跑 |
| 前日高低点假突破重新接受 | 已随机对照 |
| 前日价值区 VAH / VAL 重新接受 | 已随机对照，并进入深耕 |
| 成交量爆发边界 | 已随机对照，降级为质量标签 |
| 前日边界 + 成交量质量过滤 | 实现存在，但批量运行过慢，未作为阶段关键结论 |
| 小时等高 / 等低流动性扫单 | 已随机对照，当前方向暂停 |
| 压力释放后低波收敛再启动 | 已尝试，0 交易，不可评价 |
| 滚动日内价值区诊断 | 更适合状态分桶，不作为标准入口随机对照 |

## 3. 随机基准口径

本阶段重点使用：

```text
同事件同方向随机风险空间；
同事件随机方向；
部分宽松事件窗口随机。
```

随机方向基准不按 50% 胜率理解。短周期、明确止损、有限目标的结构入口面对的是：

```text
上涨命中目标；
下跌命中目标；
中间波动扫描止损 / 时间退出 / 成本吞噬。
```

因此，随机方向自然胜率可能只有 30% 多一点。结构胜率相对随机方向稳定高出 4 个百分点以上，即使净收益尚未显著优于随机，也可视为有研究价值的方向信息。

## 4. 阶段评价标准

结构可继续研究的标准之一：

```text
结构净收益 >= 随机分布 75% 分位
且结构最大回撤 <= 随机中位数
```

或：

```text
结构胜率相对随机方向基准稳定高出 >= 4 个百分点，
且没有被更差盈亏比、更高成本、更深回撤或亏损簇抵消。
```

或：

```text
结构净收益未转正，
但显著高于匹配随机，
且亏损簇 / 尾部亏损明显优于随机。
```

## 5. 阶段结果

随机对照阶段结论：

```text
结构入口不是整体等同随机；
价值区、前日高低点、成交量冲击均显示方向 / 胜率信息；
但原始风险空间层普遍未通过；
不同结构应分流处理，而不是统一继续做入口优化。
```

结果矩阵：

| 结构 | 随机对照结论 | 当前定位 |
|------|--------------|----------|
| 价值区 VAH / VAL 重新接受 | 随机方向优势明显，深度重新接受后风险空间改善 | 主线保留 |
| 前日高低点重新接受 | 弱方向信息，成本后仍差 | 弱线索 |
| 成交量爆发边界 | 胜率信息强但赔率极低 | 质量标签 |
| 小时流动性扫单 | 当前方向差于随机方向 | 暂停 / 反向诊断 |
| 低波收敛再启动 | 0 交易 | 暂停 |

## 6. 未完成项处理

未完成项不影响阶段核心结论：

```text
IB 无可复跑实现；
prevday_volume_filter 运行过慢；
rolling value area 是状态标签；
500/1000 seeds 目标留待正式验证。
```

本阶段不继续补齐所有基线，因为核心问题已经回答：

```text
不是所有结构都等同随机；
下一步是策略方向选择和主线聚焦。
```

## 7. 阶段决策

随机对照计划完成后的实际决策：

```text
结束铺面式结构入口随机对照；
价值区进入下一阶段主线；
成交量保留为质量标签；
前日高低点保留弱线索；
小时流动性、低波入口暂停；
未复跑结构不再阻塞阶段结题。
```
