## 1. 核心问题

本轮不再继续寻找新入口，而是补做随机基准：

```text
前日 VAH / VAL 重新接受 + POC 空间线索，
是否比匹配随机入口更稳定、更收敛？
```

如果价值区结构入口与匹配随机入口表现相近，则说明前期价值区线索可能只是事件窗口、方向偏置或风险收益空间带来的结果，而不是结构入口本身提供的信息量。

如果价值区结构显著优于匹配随机，即使暂时不能证明可交易，也说明它仍有继续做账户风险预算、MAE / MFE、失败边界质量和尾部风险补证的价值。

## 2. 本轮定位

本轮是 `pilot`，目标是打通随机对照实现和结果记录口径，不做统计显著性结论。

本轮暂不做：

```text
不做 500 / 1000 seeds 随机分布
不做多品种
不做 Walk-Forward
不做参数搜索
不据此宣布结构通过或失败
```

## 4. 随机基准实现

新增最小随机基准策略：

```text
workspace/strategies/value_area_random_baseline_strategy.py
```

策略复用 `ValueAreaReacceptanceStrategyCore` 的：

- session / value area 构造；
- POC 目标；
- strict failure / stop / time exit；
- 仓位计算；
- exit reason 诊断字段。

本轮实现了两类基准。

### 4.1 direction_matched

```text
先等待原结构重新接受事件成立；
保持原结构方向；
但用当前 bar 构造随机化失败极值，重新计算 strict distance、target distance 和仓位。
```

它回答：

```text
在同样重新接受事件和同样方向下，原结构的失败极值 / 入场定位是否优于随机化风险空间？
```

### 4.2 event_window

```text
只要当日已出现边界外突破状态，就允许在事件窗口内随机入场；
保持突破方向；
使用相同 POC 目标、止损、时间退出和成本。
```

它回答：

```text
优势是否来自价值区事件日 / 事件窗口，而不是具体重新接受入口？
```

### 4.3 direction_matched + random direction

```text
等待原结构重新接受事件成立；
但方向随机为 long / short；
仍要求 POC 目标有效，否则不入场。
```

它回答：

```text
收益是否依赖原结构方向，而不是只来自事件窗口和 POC 距离？
```

## 5. 首轮结果

### 5.1 结果汇总

| 回测 ID | 策略 / 基准 | seed | 收益率 | 净盈亏 | 夏普 | 最大回撤 | 胜率 | 交易数 | 备注 |
|---------|-------------|------|-------:|------:|-----:|---------:|-----:|------:|------|
| `397` | 结构入口 `value_area_reacceptance` | - | `+1.15%` | `+1,148.22` | `0.59` | `-1,850.74` | `44.44%` | `20` | 原结构基准 |
| `398` | `direction_matched` / same direction | `42` | `+1.84%` | `+1,837.71` | `1.00` | `-1,556.85` | `40.00%` | `22` | 单 seed 随机基准优于结构 |
| `399` | `event_window` / same direction | `42` | `-9.46%` | `-9,457.23` | `-2.11` | `-11,341.54` | `35.42%` | `102` | 事件窗口显著劣化，交易数大幅增加 |
| `400` | `direction_matched` / random direction | `42` | `-0.53%` | `-533.84` | `-0.34` | `-2,281.07` | `38.46%` | `28` | 随机方向后优势消失 |

### 5.2 直接观察

#### 观察 1：原结构明显优于宽松事件窗口随机

`event_window` 随机基准交易数从 20 增加到 102，收益率降至 `-9.46%`，最大回撤扩大到 `-11,341.54`。

这说明：

```text
价值区事件窗口本身不是优势。
边界外触发后随便交易会显著劣化。
重新接受条件至少有降噪价值。
```

#### 观察 2：原结构方向有信息量

`direction_matched + random direction` 收益率为 `-0.53%`，弱于原结构 `+1.15%`。

这说明：

```text
原结构方向不是完全随机。
VAH 上方失败做空、VAL 下方失败做多，
比同事件下随机方向更好。
```

#### 观察 3：原结构没有明显优于同事件同方向的随机化风险空间

`direction_matched / same direction` 单 seed 结果为 `+1.84%`，高于原结构 `+1.15%`，回撤也更小。

这说明当前不能说：

```text
原结构的失败极值、严格失败距离和具体入场位置，
已经显著优于同事件同方向随机基准。
```

但由于这里只跑了一个 seed，不能据此判定结构无效。

### 5.3 随机胜率基准修正

随机方向基准不应按 50% 胜率理解。

在本项目这类短周期、明确止损、有限盈利上界的结构里，随机开仓面对的不是“涨 / 跌”二元结果，而更接近：

```text
上涨命中目标
下跌命中目标
中间波动扫描止损 / 时间退出 / 成本吞噬
```

因此，随机方向自然胜率可能只有 30% 多一点。若结构入口相对随机方向基准能稳定高出 4 个百分点以上，即使净收益尚未显著优于随机，也应视为有研究价值的方向信息。

本轮 `direction_matched / random direction` 胜率为 `38.46%`，原结构胜率为 `44.44%`，单次差值为：

```text
44.44% - 38.46% = +5.98 个百分点
```

由于本轮只有单 seed，该差值不能直接定性为稳定优势，但它应成为下一轮多 seed 的重点观察指标。

## 6. 初步结论

本轮 pilot 的结论是：

```text
价值区重新接受结构没有被随机对照立即证伪，
但也没有被证明显著优于匹配随机。
```

更细分地说：

| 问题 | 当前答案 |
|------|----------|
| 价值区事件窗口是否有价值？ | 单独没有。宽松事件窗口随机显著亏损。 |
| 重新接受条件是否有价值？ | 有初步价值。它显著减少交易并改善分布。 |
| 原结构方向是否有价值？ | 有初步价值。随机方向后收益转负，胜率单次高出 `+5.98` 个百分点。 |
| 原结构具体入场 / 失败边界是否优于同方向随机？ | 未证明。单 seed 随机同方向更好。 |
| 是否可以继续研究？ | 可以，但必须进入多 seed 随机分布，不能只看单次结果。 |

## 7. 对 roadmap 的含义

这轮结果支持补充计划中的判断：

```text
结构未直接通过时，不能只问它是否转正，
还要问它是否比匹配随机更稳定、更收敛。
```

当前价值区结构暂时不能归为“彻底失去意义”，因为：

- 它明显优于宽松事件窗口随机；
- 它优于随机方向，且单次胜率优势达到 `+5.98` 个百分点；
- 它控制了交易频率和回撤；
- 但它尚未优于同事件同方向的随机化风险空间。

因此，价值区路线下一步的关键不是继续换入口，而是做：

```text
同事件同方向随机基准的多 seed 分布；
并检查原结构收益位于随机分布的哪个 percentile。
```

## 8. 下一轮计划

下一轮建议只推进一个问题：

```text
对 DCE.m2601 的 value_area_reacceptance，
运行 direction_matched / same direction 的多 seed 随机分布。
```

建议输出：

| 指标 | 目标 |
|------|------|
| seeds | 先 50，再扩展到 500 |
| 结构胜率相对随机方向优势 | 判断方向信息是否稳定高出随机方向 >= 4 个百分点 |
| 结构净收益 percentile | 判断是否高于随机中位数 / 75% / 90% |
| 结构最大回撤 percentile | 判断是否比随机更稳 |
| 亏损簇 | 判断账户生存能力是否改善 |
| 交易数分布 | 避免随机样本交易数不匹配 |
| exit reason 分布 | 判断收益来源是否相同 |

若原结构落在同方向随机分布中位数附近，且胜率相对随机方向基准不能稳定高出 4 个百分点，则价值区入口应降级为诊断变量；若能稳定超过随机 75% 分位，或方向胜率优势稳定且未被盈亏比、成本、回撤和亏损簇抵消，再进入账户风险预算和尾部风险补证。
