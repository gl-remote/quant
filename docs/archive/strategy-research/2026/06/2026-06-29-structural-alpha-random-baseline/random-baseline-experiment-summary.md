## 1. 实验问题

随机对照阶段不再问：

```text
结构入口能不能直接转正？
```

而是问：

```text
结构入口是否至少比匹配随机开仓更稳定、更收敛？
```

如果结构与随机无差异，则暂停；如果显著优于随机但仍未转正，则保留为方向线索、诊断变量或风险空间补证对象。

## 4. 关键结论

### 4.1 结构入口不是整体等同随机

价值区、前日高低点、成交量爆发边界都显示一定方向 / 胜率信息。

尤其：

```text
价值区：随机方向胜率约 34%~36%，原结构 44.44%；
成交量爆发：随机方向胜率约 56.79%，原结构 83.33%。
```

因此不能说这些结构“彻底失去意义”。

### 4.2 风险空间层普遍不足

随机对照显示：

```text
事件 / 方向可能有信息，
但当前入场价格、strict failure、止损和账户风险空间塑形没有普遍证明有效。
```

这比“入口结构失败”更精确。

### 4.3 高胜率结构的问题是赔率和左尾

成交量爆发边界胜率高，但 `win_loss_ratio` 仅约 `0.2271`。

说明：

```text
高胜率被极低盈亏比、成本和少数大亏吞噬。
```

它更适合作为状态 / 质量标签，而不是独立入口。

### 4.4 小时流动性扫单当前方向暂停

小时流动性扫单的随机方向结果显著优于原方向：

```text
原结构胜率比随机方向低约 15~16 个百分点。
```

当前方向假设不成立。

## 5. 阶段决策

| 结构 | 决策 |
|------|------|
| 价值区 VAH / VAL 重新接受 | 保留为最强方向线索 |
| 前日高低点重新接受 | 保留为弱方向线索，非优先 |
| 成交量爆发边界 | 降级为状态 / 质量标签 |
| 小时流动性扫单 | 暂停入口；可作为反向诊断候选 |
| 低波收敛再启动 | 交易机会不足，暂停 |
| 前日边界 + 成交量过滤 | 不阻塞阶段结题 |

## 6. 随机对照阶段结论

本阶段已经得到想要的核心结论：

```text
这些结构不是整体无价值；
下一步是策略方向选择，而不是继续证明是否不等同随机。
```

因此随机对照阶段结束，后续主线转向：

```text
价值区 VAH / VAL 重新接受的风险空间、账户预算、品种适配和尾部风险补证。
```
