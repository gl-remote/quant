## 1. 补充动机

当前 `structural-alpha` 多轮实验主要回答了一个问题：

```text
某个共识边界 / 接受拒绝 / 假突破 / 重新接受结构，
作为独立入口后，能否直接形成成本后正期望？
```

截至当前阶段，多数独立入口结构未通过成本后期望、账户原始盈亏比、尾部风险或跨品种稳定性标准。

但这还没有回答另一个更基础的问题：

```text
这些结构入口是否至少比随机开仓更有信息量？
```

如果结构入口的收益分布、回撤、亏损簇和收敛性与匹配随机开仓相似，则说明这些结构没有提供可验证的交易信息，应彻底暂停。

如果结构入口即使未转正，但显著优于匹配随机，则说明结构仍可能具备研究价值，下一步应转向成本空间、账户风险预算、退出结构、品种适配和尾部风险补证，而不是继续寻找新入口。

## 2. 核心问题

本补充计划优先回答：

```text
已测试的结构入口，
在同品种、同时间窗口、同交易频率、同成本、同退出规则、同风险空间下，
是否比随机开仓产生更稳定、更收敛、更可解释的收益分布？
```

核心不是判断结构是否已经可上线，而是判断：

```text
结构入口是否显著区别于随机位置？
```

## 3. 原假设与备择假设

### 3.1 原假设

```text
H0：结构入口与匹配随机入口没有显著差异。
```

表现为：

- 结构净收益位于随机分布中位数附近；
- 结构最大回撤、亏损簇和尾部亏损不优于随机；
- 结构胜率 / 盈亏比 / 成本后期望不优于随机；
- 结构结果主要由少数交易或单一品种贡献；
- 多次随机种子下，结构优势不稳定或不可复现。

### 3.2 备择假设

```text
H1：结构入口虽然未必直接转正，但显著优于匹配随机入口。
```

表现为：

- 结构净收益位于随机分布较高分位；
- 结构最大回撤小于随机中位数或低分位风险更好；
- 结构亏损簇更短、更浅；
- 结构成本后胜率、盈亏比或盈亏平衡胜率安全边际优于随机；
- 结构在多个品种、窗口或参数邻域中稳定优于随机；
- 结构收益来源可以解释为共识边界、接受 / 拒绝质量、失败边界、盈利上界或账户风险空间，而不是随机尖峰。

## 4. 对照对象

第一轮随机对照优先覆盖已经完成主要实验的结构路线：

| 结构路线 | 对照目的 |
|----------|----------|
| Initial Balance 假突破重新接受 | 判断 IB 重新接受是否只是日内随机反转噪声 |
| 前日高低点假突破重新接受 | 判断跨天高频可见边界是否优于同窗口随机 |
| 前日价值区 VAH / VAL 重新接受 | 判断 POC 空间线索是否显著优于随机 |
| 成交量爆发边界 | 判断 volume shock 作为主边界是否有超随机信息 |
| 前日边界 + 成交量质量过滤 | 判断 breakout shock 过滤是否优于随机降频 |
| 小时等高 / 等低流动性扫单 | 判断小时流动性池是否优于随机扫单位置 |
| 压力释放后低波收敛再启动 | 判断低波再启动是否只是普通短周期突破噪声 |
| 滚动日内价值区诊断 | 判断滚动价值区标签是否能改善随机基准分布 |

本轮不新增入口结构，不做 Optuna，不做 Walk-Forward，不通过调参追求转正。

## 5. 随机基准类型

每个结构至少建立以下随机基准。

### 5.1 同交易窗口随机开仓

保持：

```text
同品种
同回测区间
同交易时段
同交易次数
同持仓周期约束
同手续费和滑点
同止盈 / 止损 / 时间退出规则
```

仅随机化入场时间。

回答：

```text
结构入口是否优于同样交易频率下的随机入场？
```

### 5.2 同方向比例随机开仓

在同交易窗口随机开仓基础上，保持原结构多空比例。

例如原结构样本为：

```text
long 60%
short 40%
```

随机基准也使用相同比例。

回答：

```text
结构收益是否只是来自多空方向偏置？
```

### 5.3 同风险收益空间随机开仓

随机样本需要匹配原结构的风险收益空间，至少包括：

```text
strict_distance 分布
expected_target_distance 分布
max_hold_bars
stop_widen_multiplier
实际止损距离
时间退出规则
```

回答：

```text
在相同失败距离和盈利上界条件下，结构入口是否仍优于随机？
```

这是最重要的基准，因为结构型 alpha 的核心不是“任意位置开仓”，而是“在明确失败边界和盈利上界下验证方向选择”。

### 5.4 同事件日 / 同事件窗口随机开仓

只在结构事件出现的日期或窗口内随机开仓。

例如：

```text
出现 VAH / VAL 重新接受事件的交易日
出现前日高低点假突破事件的交易日
出现 IB 假突破事件的日内窗口
出现 volume shock 的后续有效窗口
```

但随机选择具体入场 bar 或方向。

回答：

```text
优势来自事件日市场状态，还是来自具体结构入口？
```

### 5.5 随机方向胜率基准

随机方向基准不能默认按 50% 胜率理解。

本项目关注的是短周期、明确止损、有限盈利上界的结构入口。一次随机开仓面对的结果通常不是简单二元：

```text
上涨命中目标
下跌命中目标
中间波动扫描止损 / 时间退出 / 成本吞噬
```

因此，随机方向的自然胜率可能只有 30% 多一点，而不是 50%。

这意味着：

```text
结构胜率相对随机方向基准稳定高出 4 个百分点以上，
即使净收益尚未显著优于随机，
也应视为有研究价值的方向信息。
```

但该胜率优势不能单独作为通过依据，还必须继续检查：

- 成本后盈亏比是否被压低；
- 盈亏平衡胜率是否仍明显高于实际胜率；
- 胜率优势是否被更深回撤或亏损簇抵消；
- 优势是否只来自单 seed、单品种或少数交易。

## 6. 匹配规则

随机对照必须避免不公平比较。

### 6.1 必须保持一致的项目

| 项目 | 要求 |
|------|------|
| 品种 | 与结构实验一致 |
| 数据区间 | 与结构实验一致 |
| K 线周期 | 与结构实验一致 |
| 交易时段 | 与结构实验一致，避免夜盘跨段污染 |
| 成本口径 | 手续费、滑点、合约乘数一致 |
| 交易次数 | 随机样本应匹配结构样本数量，或报告交易数差异 |
| 多空比例 | 至少有一组随机基准匹配结构多空比例 |
| 持仓周期 | 与结构实验 `max_hold_bars` / `force_flat_time` 一致 |
| 止盈止损 | 与结构实验退出规则一致 |
| 风险收益空间 | 至少有一组随机基准匹配 strict distance 和 target distance 分布 |

### 6.2 不允许的对照方式

以下对照不作为有效证据：

- 用全天任意随机开仓对比只在高质量事件窗口出现的结构；
- 用不同交易次数比较净收益；
- 用不同止损、止盈或时间退出比较结构优势；
- 用不含成本的随机结果对比含成本结构结果；
- 随机样本没有固定 seed，导致结果不可复现；
- 只挑选一次随机结果，不报告随机分布。

## 7. 随机次数与输出分布

每个结构、每个品种、每个随机基准类型，至少输出一组随机分布。

建议第一轮：

```text
random_seeds = 500
```

若计算成本可接受，再提升到：

```text
random_seeds = 1000
```

每个随机分布至少记录：

| 指标 | 用途 |
|------|------|
| `random_net_pnl_distribution` | 判断结构收益分位 |
| `random_win_rate_distribution` | 判断胜率是否优于随机 |
| `random_profit_factor_distribution` | 判断盈亏比结构是否改善 |
| `random_max_drawdown_distribution` | 判断回撤是否更稳定 |
| `random_max_single_loss_distribution` | 判断尾部单笔亏损 |
| `random_loss_cluster_distribution` | 判断连续亏损和账户生存能力 |
| `random_cost_to_avg_profit_distribution` | 判断成本吞噬程度 |
| `random_exit_reason_distribution` | 判断收益来源是否不同 |

## 8. 评价指标

结构结果需要放入随机分布中评价。

| 指标 | 解释 |
|------|------|
| 净收益分位 | 结构 `net_pnl` 位于随机分布的 percentile |
| 回撤分位 | 结构 `max_drawdown` 相对随机是否更浅 |
| 亏损簇分位 | 结构最大连续亏损是否优于随机 |
| 成本后胜率分位 | 结构胜率是否优于随机 |
| 随机方向胜率优势 | 结构胜率 - 随机方向基准胜率；稳定高出 4 个百分点以上说明方向信息有研究价值 |
| 成本后盈亏比分位 | 结构平均盈利 / 平均亏损是否优于随机 |
| 盈亏平衡安全边际差 | 结构胜率安全边际是否高于随机中位数 |
| 收敛性 | 随机分布是否很宽，结构是否稳定落在优势区域 |
| 多品种一致性 | 结构是否在多个品种上均优于随机 |
| 多窗口一致性 | 结构是否在不同时间窗口优于随机 |

## 9. 通过、保留与暂停标准

### 9.1 结构可继续研究

满足以下条件之一，可继续作为结构线索研究：

```text
结构净收益 >= 随机分布 75% 分位
且结构最大回撤 <= 随机中位数
```

或：

```text
结构胜率相对随机方向基准稳定高出 >= 4 个百分点，
且没有被更差盈亏比、更高成本、更深回撤或亏损簇抵消。
```

或：

```text
结构净收益未转正，
但显著高于匹配随机，
且亏损簇 / 尾部亏损明显优于随机。
```

这类结构不能直接进入上线验证，但可以进入下一步：

- 成本空间补证；
- 账户风险预算补证；
- 失败边界质量诊断；
- MAE / MFE 分桶；
- 品种适配；
- 参数邻域稳定性。

### 9.2 结构仅保留为诊断变量

若结构不显著改善净收益，但能稳定改善某类诊断指标，例如：

- 降低最大回撤；
- 缩短亏损簇；
- 降低 strict failure 频率；
- 改善成本 / 平均盈利；
- 改善特定品种或特定上下文；

则不作为独立入口，但可以保留为状态变量、上下文标签或事后分桶变量。

### 9.3 结构暂停

满足以下情况，应暂停当前结构入口：

```text
结构净收益位于随机分布 50% 附近或更差；
结构回撤、亏损簇、尾部亏损不优于随机；
结构优势只来自单一随机尖峰、单一品种或极少数交易；
同风险收益空间随机后优势消失；
同事件日随机后优势消失。
```

若所有结构入口均无法显著优于匹配随机，则当前“共识边界直接入口”路线整体暂停。

## 10. 对当前短期计划的影响

本补充计划不替代原短期计划，而是在其后增加一个必要判定层：

```text
结构入口是否成立
→ 若未直接通过
→ 先与匹配随机入口对照
→ 若不优于随机，暂停结构
→ 若优于随机但未转正，进入风险预算 / 成本 / 尾部风险补证
```

因此，后续不应继续只问：

```text
这个结构能不能直接赚钱？
```

还必须问：

```text
这个结构是否比随机开仓更稳定、更收敛、更少尾部风险？
```

## 11. 优先执行顺序

建议按以下顺序执行随机对照：

```text
1. 前日价值区 VAH / VAL 重新接受 + POC 空间线索
2. 前日高低点重新接受
3. Initial Balance 重新接受
4. 成交量爆发边界
5. 前日边界 + 成交量过滤
6. 小时等高 / 等低流动性扫单
7. 低波收敛再启动
8. 滚动日内价值区标签
```

优先价值区方向的原因：

```text
它是当前唯一出现“子样本显著改善”的结构线索；
如果它也不能优于匹配随机，则当前阶段结构型入口研究应整体收束。
```

## 12. 第一轮产出物

第一轮随机对照至少产出：

| 产出物 | 说明 |
|--------|------|
| 随机对照实验记录 | 写入 `docs/workbench` |
| 结构 vs 随机分布表 | 每个结构、品种、基准类型的分位结果 |
| 随机基准实现说明 | 随机 seed、匹配规则、排除规则 |
| 显著优于随机的结构列表 | 可继续研究的结构线索 |
| 与随机无差异的结构列表 | 暂停或归档 |
| 诊断变量列表 | 不作为入口但可保留的状态变量 |
| 阶段结论 | 当前结构型入口路线继续、收缩或暂停 |

## 13. 阶段决策规则

随机对照完成后，按以下方式更新短期研究方向：

```text
若所有结构 ≈ 匹配随机
→ 当前结构型入口路线暂停，归档本阶段。

若少数结构显著优于随机但仍成本后为负
→ 不继续找新入口，转向账户风险预算、成本空间、失败边界质量和尾部风险补证。

若价值区方向显著优于随机
→ 将下一阶段聚焦为“价值区 POC 空间 + 品种适配 + 风险预算补证”。

若多个结构显著优于随机
→ 优先选择分布更稳定、亏损簇更小、账户风险更可执行的结构，而不是净收益最高的结构。
```

最终目标仍与长期框架一致：

```text
不是寻找更准的预测信号，
而是确认某些共识价格区间是否真的比随机位置更适合作为低成本验证方向选择的交易场。
```
