## 首轮的方法论关键发现

### KF-B1 · 波动率变化率类因子与未来漂移强度独立

- 类型：方法论
- 状态：已证实（F1 + F3 两条独立证据链）
- 证据：`candidate-f1-atr-turning-20-60.md` + `candidate-f3-realized-vol-breakout-20-60.md`
- 影响：Wave 2 融合时不应把 ATR / RV 变化率作为强度权重 · 只可作为覆盖率保护条件
- 直觉解释：GBM 下 σ 变化并不改变 ν 的期望 · 波动率突破改变 barrier 触达时间但不改变方向偏好

### KF-B2 · 长程记忆（Hurst）与短期漂移强度独立

- 类型：方法论
- 状态：已证实（F5）
- 证据：`candidate-f5-hurst-60.md`
- 影响：Hurst 描述的是**过去** 60h 的时间序列结构强度 · 与**未来** 20h 的具体漂移强度无直接映射
- 直觉解释：Hurst > 0.5 只保证过去有 mean-reverting < random walk 的 persistence · 不保证未来出现强漂移

### KF-B3 · 窗口回归 se 上限证伪（实测验证）

- 类型：方法论 + 假设证伪
- 状态：已证实（F11 · 实测 se_hat = 0.2394，与理论预测 $1/\sqrt{20}$ = 0.224 相差 6.9%）
- 证据：`candidate-f11-window-regression-20h.md` · shaping-theory §2.23.5.5
- 影响：任何"直接从历史窗口估计 $|\nu|/\sigma$"的因子都会被 Gate 1 拒绝 · 未来必须用 non-window 结构

### KF-B4 · Gate 1.5 揭示"分布对齐 ≠ 逐点预测"

- 类型：方法论
- 状态：已证实（F11 特例）
- 证据：F11 的 Gate 1.5 完美通过（C1-C4 全过）但 Gate 3 秩相关 r = -0.082 近似零
- 影响：Gate 1.5 与 Gate 3 是**正交检验** · Gate 1.5 只测边缘分布 · Gate 3 才测逐点相关 · 二者都必要

## 首轮方向推断

**四条独立证据链得到同一个结论**：**Wave 1 候选（全部基于波动率变化 / 长程记忆）都无法预测未来漂移强度**。

这在方法论上强化了 Wave 2 的方向选择：

- **候选**：能预测**未来**漂移的信号（不是描述**过去**波动的）
- **例如**：event-driven 信号（政策 / 库存 / OPEC · shaping-theory §5.5 已列）
- **横截面共振**：同板块 ≥3 品种齐涨齐跌（本质是识别"当下正在形成的漂移"而非"过去的记忆"）

**Wave 1 的教训将写入 research-status.md 的 KF 清单**。

**方向证据**：过去 20-60h 内出现强漂移 / 高成交量的品种，未来 20h 的漂移强度**略微偏低**（负相关很弱但方向一致）——弱均值回归特征。

## Wave 3 · 内生 + 制度切换 全面收尾

### Wave 3 结果汇总

Wave 3 测试了两个因子，全数失败：

| 因子 | 类型 | se_ratio | r_hat | 级别 |
|---|---|---|---|---|
| F13 成交量突增 | 内生 · 成交量 MDH | 5.27 | -0.126 | L4 |
| F14 交易时段效应 | 制度切换 · 日内时段 | 3.61 | 0.046 | L4 |

这是**第 7-8 个**跨类型的失败因子：

| 因子类型 | 代表因子 | 结果 |
|---|---|---|
| 窗口回归型 | F11 | L4 · r=-0.082 |
| 长程记忆型 | F5 | L4 · r=0.061 |
| 波动率变化型 | F1 / F3 | L4 · r∈[-0.067, -0.003] |
| 横截面型 | F7 / F7-alt | L4 · r∈[-0.117, -0.094] |
| 成交量型 | F13 | L4 · r=-0.126 |
| 制度切换型 | F14 | L4 · r=0.046 |

**特别值得注意的 F14**：这不是"过去预测未来"型因子，而是"制度标签"型因子——如果时段效应存在，它本应有预测力（因为未来窗口就在该时段内）。但实测三个时段的均值差仅 1.5%，几乎完全一致。这说明**玉米 1h 上连"制度切换"型的强度差异都不存在**。

### 结论：内生 + 制度切换路径彻底穷尽

所有可从价量数据直接提取的信息类型都已测试：
- ✅ 时序统计（窗口回归 / Hurst / ATR / RV）
- ✅ 横截面（方向共振 / 平均强度）
- ✅ 成交量（MDH 假设）
- ✅ 制度切换（日内时段）

**全部 8 个候选，无一存活**。Gate 3 秩相关范围 [-0.126, 0.061]，平均 -0.048。
