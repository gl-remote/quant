## 1. 实验问题

R5/R6 从成交记录和派生字段里看到：

```text
POC 靠边是风险信号；
target_to_va 过大不等于更好盈亏比；
POC / VA 真正有效时，提供的是较近失败边界、适中可兑现目标和足够原始盈亏比。
```

本轮不调参、不新增回测，改做 DCE.m2601 关键样本的 K 线路径复盘，回答：

```text
1. 盈利样本里的 POC 是否真的像可回归共识锚；
2. 亏损样本里的 POC 是否只是历史成交密集点，已经不再可兑现；
3. VAH / VAL 是否在图形路径上像清晰失败边界；
4. 2~3 ticks 敏感性是否可以被 POC / VA 结构脆弱性解释。
```

## 2. 数据来源

使用 R5/R6 的 5m 样本：

```text
DCE.m2601:
backtest_id = 401, min_reaccept_ticks = 2
backtest_id = 402, min_reaccept_ticks = 3
```

数据来源：

```text
trade_clearings：逐笔清算结果；
backtest_trades.decision_payload_json：入场时 VAH / VAL / POC / target / strict_failure；
project_data/market_data/csv/DCE.m2601.tqsdk.5m.csv：复盘入场日前后 5m K 线路径。
```

解释成交时间时需要注意：

```text
trade open_time 是下一根 bar 的成交时间；
entry price 对应上一根 5m bar 的信号收盘价 / 下一根开盘成交价。
```

因此，本轮关注路径结构，不把单根 bar 的时间戳差一格作为策略逻辑差异。

## 4. 盈利样本复盘

### 4.1 2025-10-22 long：最接近理想结构

前一日结构：

```text
VAL = 2860
VAH = 2899
POC = 2880
POC 位于 VA 中部附近，poc_pct = 0.513
entry 到 POC 占 VA 宽度 43.6%
raw_rr = 1.889
```

当日路径：

```text
09:00 附近价格直接在 VAL 附近波动；
09:10~09:20 下破 VAL 后快速收回；
09:25 成交在 2863；
10:05~10:10 价格上冲到 2883/2885，完成 POC 回归。
```

诊断：

```text
这是当前最标准的 value_area_reacceptance 形态：
1. 失败边界靠近 entry；
2. POC 不靠边；
3. entry 到 POC 有空间，但不是过远目标；
4. 回归路径发生得很快，没有依赖长时间持仓。
```

这里 POC 更像“可回归共识锚”，不是单纯远目标。

### 4.2 2025-09-29 short：上破 VAH 后失败，POC 可兑现

前一日结构：

```text
VAL = 2930
VAH = 2948
POC = 2935
POC 偏下但未极端靠边，poc_pct = 0.278
```

当日路径：

```text
13:30 左右上破到 2952；
随后 13:50~14:00 收回 VAH 内侧；
14:35~14:45 快速下行到 2936 / 2933 附近，触发 POC 目标。
```

诊断：

```text
这笔不是最理想的 POC 居中形态，
但失败动作清楚，且 POC 靠近当日下午回落路径上的真实接受区，
所以 2 ticks 和 3 ticks 都能兑现。
```

2 ticks 的 entry 更高，raw_rr 更好；3 ticks 延后入场后仍盈利，但盈利空间被压缩。这说明：

```text
ticks 深度的作用更像“等待失败确认”，
不是越深越好。
```

### 4.3 2025-11-12 short：盈利但不是干净 take_profit 样本

前一日结构：

```text
VAL = 3044
VAH = 3063
POC = 3052
POC 不靠边，poc_pct = 0.421
entry 到 POC 占 VA 宽度 47.4%
```

当日路径：

```text
14:05 附近上冲到 3073；
14:25~14:30 收回到 VAH 内侧；
21:00 force_flat 附近退出盈利。
```

诊断：

```text
POC / VA 结构本身较合理，
但该样本以 force_flat 退出，不是标准 POC hit。
```

因此它可以支持“目标距离适中更好”的判断，但不能作为强证据证明 POC 回归稳定。

## 5. 亏损样本复盘

### 5.1 2025-09-24 short：当前日接受区已经上移

前一日结构：

```text
VAL = 2911
VAH = 2936
POC = 2917
poc_pct = 0.240
target_to_va = 0.600
```

当日路径：

```text
上午价格从 2912 附近逐步抬升到 2930 上方；
13:30 后在 2930~2938 区域成交活跃；
short 入场后没有向 2917 回归，time_exit 基本持平；
夜盘随后跳到 2950 以上，说明日内接受区已明显上移。
```

诊断：

```text
这不是清晰的“上破 VAH 失败后回归 POC”，
更像旧 VA 上沿附近形成新的接受区。
```

前日 POC 虽然不是极端靠边，但已经明显低于当前日主成交路径。此时 POC 是历史锚，不是短期可兑现目标。

### 5.2 2025-10-15 short：POC 偏低，目标路径过长

前一日结构：

```text
VAL = 2889
VAH = 2913
POC = 2894
poc_pct = 0.208
target_to_va = 0.667
raw_rr = 0.762
```

当日路径：

```text
上午价格从 2926 附近缓慢下行；
13:50 在 VAH 内侧入场 short；
之后没有继续下行到 POC，反而回到 2917 附近 time_exit。
```

诊断：

```text
这笔更像日内下行后进入前日价值区上沿，
不是区间外价格被快速拒绝。
```

POC 靠近 VA 下沿，entry 到 POC 距离占 VA 比例偏大，而且 raw_rr 本身不强。这里失败不是 ticks 太浅，而是：

```text
POC 目标不够可兑现；
VAH 也不像清晰失败边界。
```

### 5.3 2025-11-05 short：POC 极靠边，旧目标失效

前一日结构：

```text
VAL = 3011
VAH = 3029
POC = 3013
poc_pct = 0.111
target_to_va = 0.722
raw_rr = 1.625
```

当日路径：

```text
09:00 后价格快速上冲到 3033；
09:35 入场后，价格长时间维持在 3024~3032 附近；
没有向 3013 回归；
午后和夜盘继续大幅上行到 3070 / 3080 以上。
```

诊断：

```text
这是 POC 靠边风险最清楚的样本。
```

虽然 raw_rr 看起来不差，但 target 在 VA 下沿附近，已经远离当前日真实接受区。此时：

```text
POC 不是“更好盈亏比”，而是“更远且不可兑现的历史目标”。
```

## 6. 本轮关键发现

### 6.1 POC 是否可兑现，比 POC 距离更重要

盈利样本并不是 POC 越远越好，而是：

```text
POC 位于可回归路径中；
entry 到 POC 的距离适中；
当前日没有形成明显背离旧 POC 的新接受区。
```

亏损样本常见问题是：

```text
POC 靠边；
target_to_va 偏大；
当前日成交已经迁移到旧 VA 的另一侧或外侧；
价格没有理由在短持仓窗口内回到旧 POC。
```

### 6.2 VAH / VAL 不是天然失败边界

VAH / VAL 在理想样本里有效，是因为它们同时满足：

```text
先发生区间外突破；
随后快速重新接受；
失败边界靠近 entry；
POC 仍在短期可到达路径内。
```

但在亏损样本中，VAH / VAL 更像历史成交密集区边缘：

```text
如果当日已经在 VA 上方形成新的接受，
再收回 VAH 内侧并不一定表示失败，
可能只是新旧价值区之间的过渡。
```

### 6.3 2~3 ticks 是经验缓冲，不是核心 edge

R7 支持 R6 的判断：

```text
2~3 ticks 可以减少刚贴边收回的噪声，
但不能修复 POC 靠边、目标过远、当前日接受区迁移这些问题。
```

在 2025-10-22 这类好结构里，浅确认已经足够；
在 2025-11-05 这类差结构里，即使确认更深，也只是更晚进入一个不可兑现目标。

### 6.4 close-profile POC 有解释力，但还不够严格

当前 close-profile POC 能解释一部分样本：

```text
POC 居中、target_to_va 适中时，样本质量更好；
POC 靠边、target_to_va 过大时，风险明显升高。
```

但它仍然不够严格：

```text
单点 POC 无法表达 POC 周围的接受带；
前日 POC 可能在当前日快速失效；
多峰或迁移型分布下，单一 POC 可能不是共识锚。
```

## 7. 阶段结论

R7 将 R5/R6 的统计形态线索落实到了 K 线路径上：

```text
value_area_reacceptance 的有效性不来自“POC 更远、raw_rr 更高”，
而来自旧 VA 边界被快速拒绝后，
价格仍有较高概率回到一个距离适中、位置合理、可兑现的 POC / POC band。
```

当前最重要的结构判断是：

```text
1. POC 是否远离 VA 边缘；
2. entry 到 POC 是否适中，而不是过远；
3. 当前日是否已经形成背离旧 POC 的新接受区；
4. VAH / VAL 是否真的表现为失败边界；
5. 回归是否能在短持仓窗口内发生。
```

因此，后续仍不应急于调参。更合理的下一步是：

```text
继续用关键样本验证 POC / VA 定义，
优先比较 close-profile、range-profile、POC band 或内部目标区是否更能表达“可兑现共识锚”。
```
