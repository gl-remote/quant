## 2. ✅ 正方证据链：子假设**成立**

### 证据链 ①：反转土壤满足 — 必要条件通过

> **逻辑**：VA reaccept 的本质是「价格触及价值区边界后向中枢回归」。若价格存在强正自相关（趋势延续），回归机制失效。反转土壤是 reaccept 策略成立的前置必要条件。

- 全局池自相关高达 **+0.851**（相邻 8h 窗口方向高度一致），意味着在全样本环境中，reaccept 会被趋势持续打脸 — 这也正是旧 VA reaccept 整体被冻结的核心原因。
- **三维子池自相关降至 +0.680**，降幅 **−0.171**。虽然仍是正自相关（没有到纯均值回归的负域），但趋势延续性显著削弱，反转行为的「生存空间」被打开。
- 判据 ③（子池自相关 < 全局）**通过**。

### 证据链 ②：统计显著性极强 — 双口径 CI 完全排 0

> **逻辑**：若 reaccept 只是 noise，配对差应围绕 0 震荡；若存在真实增量，CI95 应系统性不包含 0。

- 扁平成本：CI95 = **[+29.4, +63.6] bps**，下限距 0 有 29+ bps 缓冲，`p(diff ≤ 0) = 0.000`（200 次 bootstrap 中无一次出现非正差值）。
- 真实成本：CI95 = **[+28.9, +67.3] bps**，结论同构，成本口径切换对 CI 分布几乎无影响（形状不变、中心仅下移 0.7 bps）。
- 判据 ①（CI 排 0）**双口径均通过**。

### 证据链 ③：泛化性达标 — 超 2/3 合约方向一致

> **逻辑**：品种保留率 < 50% 意味着「只是少数品种碰巧贡献了正差值」，不可泛化；≥ 60% 才说明增量具有跨品种的普适性。

- 扁平成本下 **71.8% 合约**（79/110）的 reaccept 配对差为正；
- 真实成本下 **66.4% 合约**（68/102）保持正差值；
- 即使切换到更苛刻的成本口径，仅损失 5.4 个百分点的保留率，**无结构性崩塌**。
- 判据 ②（品保 ≥ 60%）**双口径均通过**。

### 证据链 ④：触发器具有「保护型 alpha」属性

| 组 | 真实成本 均值 | 命中率 |
|----|---------------|--------|
| reaccept（触发后做同方向） | **+1.74 bps** | 46.8% |
| no_trigger（同池内同方向随机做） | **−44.49 bps** | 32.9% |
| Δ | **+46.23 bps** | +13.9pp |

- 子池内「裸做同方向」（no_trigger）显著亏损（−44 bps / 8h），说明三维子池本身对方向判断是「有毒」的 — 在此环境中**随意开仓 = 稳定亏钱**。
- reaccept 触发器将结果拉回正值区间，意味着它提供的不是「预测方向的 alpha」，而是**「避开有毒时点 + 选择回归窗口」的保护型 alpha**。
- 这种 alpha 在实际组合中可与 poc-value-area-asymmetry 分类器（偏方向 alpha）形成**互补**：分类器抓倾斜方向，reaccept 抓对称环境下的回归窗口。

## 3. ❌ 反方证据链：子假设**不成立**

### 证据链 ①：绝对收益太薄 — 真实成本下仅 +1.74 bps/8h

> **逻辑**：CI 排 0 ≠ 可交易。策略最终需要以「可覆盖运营成本 + 资金成本 + 意外滑点」的绝对收益为底线。

- 真实成本口径下，reaccept 单事件净收益 **+1.74 bps**。按 8h 持有期估算：
  - 假设每个合约每天触发 ~0.6 次（474 次 / 140 合约 / 322 天 × 触发率 11.6% → 实际上更稀），
  - 年化 = 1.74 bps × 250 × 0.6 ≈ **2.6%**（未计名义暴露压缩 / 保证金杠杆后可放大，但夏普会下降），
  - 对比基线策略 Sharpe 2.23 / 年化 15.45%，该增量在组合层面贡献度不足 **20%**。
- **更致命**：命中率仅 **46.8%**，说明策略赚钱靠的是盈亏比而非胜率 — 但当前 holding=8h 下，单次 +1.74 bps 的均值对应的盈亏结构非常脆弱，任何执行摩擦（真实滑点模型升级、T+1 限仓等）都可能把它打回负值。

### 证据链 ②：触发器代理过粗 — 结论可能是「rank20 代理有效」≠「VA reaccept 有效」

> **逻辑**：Gatekeeper 用的是 rank20 + close_diff 的最简代理，与真实 VA reaccept 检测器有本质差异。

| 要素 | 本实验代理 | 严格 VA reaccept（旧策略定义） |
|------|-----------|-------------------------------|
| VA/POC 冻结 | ❌ 完全没有，未计算任何 VA 统计 | ✅ 前一 session POC/VAH/VAL 冻结，不得实时漂移 |
| 「回归中枢」定义 | ❌ 隐式假设（rank20 本身即中枢） | ✅ 显式以 POC 或 VA 中点为锚 |
| 「触及 VA 边界」 | ❌ 无边界概念，rank20 ≤ 20% 即近似 | ✅ 价格必须先跌破 VAL 或涨破 VAH |
| 「re-enter 重入」 | ❌ 反弹/回落即算 | ✅ 破边界后必须重新回到 VA 内部才算 reaccept |
| 时间窗精度 | ❌ 1h event 级，边界不可区分 | ✅ 5m bar 级严格逐 bar 检测 |

- **反方核心攻击点**：实验中表现出来的 +46 bps 配对差，**可能只是「在 VA 对称环境中，20 期 rank 的反转效应」**，和传统「Auction Market Theory 下 VA reaccept 的机制」无因果关系。
- 若用严格检测器重跑，极可能出现：
  - 触发样本量下降 70%+（严格条件更苛刻），
  - 配对差 CI 收缩甚至包含 0，
  - 结论反转。

### 证据链 ③：子池本身偏空 — no_trigger 组 −44 bps 可能是方向性偏差而非 realpha 增量

- 制度土壤显示：子池 ret_8h 均值 = **−1.08 bps**，虽接近 0，但方向为负。
- no_trigger 组在 flat 成本下 −47.58 bps / real −44.49 bps，**显著差于子池无条件均值（−1.08 bps）**，说明 no_trigger 的「同方向匹配」可能引入了额外的选择偏差：
  - 对 L 侧 reaccept，no_trigger 匹配的是「类似反弹幅度」的事件，但这些事件可能恰好分布在 rank20 偏高的位置（因为池子里 rank20 低 + close_diff > 0 的样本天然稀缺，只能匹配到 rank20 中部的反弹），而 rank20 中部做多 = 做趋势中继，在对称子环境的弱反转土壤下就是亏钱的。
- **反方解释**：配对差的 46 bps 增量，至少有一半来自「no_trigger 组被人为选成了亏钱样本」，而不是 reaccept 组本身真的有 alpha。
- 需补做安慰剂验证：把 reaccept / no_trigger 标签随机打乱重跑配对，看是否仍出现正配对差（若仍出现，则配对设计有系统性偏差）。

### 证据链 ④：趋势平稳的近似定义有漏洞

- 本实验用 `trend_rank_roll ∈ [0.20, 0.80]`（中间 60%）近似「趋势平稳」，但：
  - `trend_rank_roll` 本身是 rolling rank 值，**中间 60% ≠ 横盘**，可能包含温和上涨 / 温和下跌（rank 0.5 附近的趋势实际上就是「稳定的趋势」），
  - 严格版需要 `transition_flag` 检测（趋势 → 横盘 / 横盘 → 趋势的时点），仅在横盘维持期内算「趋势平稳」，
  - 严格版触发池可能只有当前的 **30-50%**，样本量进一步收缩后，bootstrap 稳定性存疑。

### 证据链 ⑤：泛化性不足 — 仍有 33-38% 合约持续亏损，且 3 合约完全无法触发

- 真实成本下 33.6% 合约（34/102）的配对差为负，这不是「尾部噪音」（按 68-95-99.7 法则，1σ 应只有 16% < 0），**说明至少有一个显著的子群体对该策略天然免疫**。
- 3 个合约在三维子池内连一次 reaccept 触发都没有，泛化性并非「全域 140 合约都好用」，而是「110 个合约能触发，其中 68 个合约真好用」— 覆盖率仅 **48.6%（68/140）**，不足一半。
- 若下游组合需要品种均匀配置，这个覆盖率是硬伤。

### 证据链 ⑥：Cluster bootstrap B=200 定性够用，但精度不足

- B=200 用于 Gatekeeper 快速 yes/no 判断是合适的，但 CI95 上下限的分位数估计误差约为 `±1/sqrt(200) ≈ ±7%`，
- 若下放到正式实验，必须 B≥2000 才能确保 CI95 的稳定估计；当前 CI 宽度虽不包含 0，但**收缩后是否仍排 0 存在不确定性**。

### 反方综合评分

| 否决条件（命中任一即不成立） | 结果 |
|--------------------------|------|
| 绝对收益太薄（< 5 bps/8h） | ❌ **命中（+1.74 bps）** |
| 代理 ≠ 机制（可能只是 rank20 反转） | ❌ **命中（未验证严格检测器）** |
| 配对设计偏差（no_trigger 天然偏负） | ❌ **命中（缺安慰剂验证）** |
| 趋势平稳定义漏洞 | ❌ 命中（缺 transition_flag） |
| 品种覆盖率不足一半 | ❌ **命中（48.6%）** |
| Bootstrap 精度不足（B<2000） | ❌ 命中（B=200） |
| **合计** | **6/6 全命中** |

## 4. ⚖️ 综合判决

### 4.1 正反双方权重比较

| 维度 | 正方得分 | 反方得分 | 胜出方 |
|------|----------|----------|--------|
| **机制合理性** | 反转土壤通过，但机制代理未验证 | 代理 vs 机制差异 + 安慰剂缺失 | 反方（机制链条断裂） |
| **统计显著性** | CI 双口径排 0，p=0.000，证据强 | B=200 精度不足，但不影响定性 | 正方 |
| **绝对可交易性** | 真实成本转正，方向正确 | 太薄（+1.74 bps）+ 低胜率（46.8%） | 反方 |
| **泛化性** | 品保 66%+ 达标 | 品种有效覆盖率仅 48.6% | 平局 |
| **组合贡献度** | 与方向型分类器互补 | 年化贡献 < 20%，边际效益低 | 反方 |
| **执行稳健性** | 成本口径切换影响极小 | 执行摩擦可能把净收益打负 | 反方 |
| **合计** | **1 胜 1 平** | **4 胜 1 平** | **反方 4:1 胜出** |

### 4.2 最终判决

> **⚠️ 判决：NOT READY FOR STAGE 1 — Gatekeeper 定性通过但机制层不成立，需降级为「条件性探索支线」**

- **Gatekeeper 统计层面**：✅ **PASS**（5/5 硬判据全通过，CI 排 0 + 品保达标 + 反转土壤成立）
- **机制与落地层面**：❌ **REJECT**（6/6 否决条件命中，核心问题是「代理 ≠ 真实 VA reaccept」+「绝对收益太薄不可交易」）

### 4.3 可执行的后续路径（三选一）

| 路径 | 动作 | 预计代价 | 适用情况 |
|------|------|----------|----------|
| **A 严格验证** | 重写严格 VA reaccept 检测器（5m bar + POC 冻结 + transition_flag），B=2000，安慰剂 shuffle | 中（~1 人日脚本 + ~1 小时算力） | 要把该策略作为「独立 alpha 源」纳入组合 |
| **B 降级为辅助信号** | 不严格验证机制，直接把 rank20 代理作为对称子环境的「辅助 gate / 保护信号」嵌入现有分类器 pipeline | 低（~半天） | 要的是 rank20 反转的保护型 alpha，不在乎名字是不是 VA reaccept |
| **C 暂停搁置** | 归档结果，下游组合仅保留分类器 v4.0 + 塑形基准，不引入对称子环境支线 | 无 | 要聚焦主线，避免分散精力 |

**推荐路径：B**（降级为辅助信号）。理由：
1. 机制虽未被严格验证，但统计显著且跨成本稳定，**浪费掉 46 bps 的配对保护增量不划算**；
2. 作为辅助 gate（例如在对称子环境中，要求分类器信号同时满足 rank20 同侧条件才开仓），不需要严格证明它是「VA reaccept」，只要它是有效的保护特征即可；
3. 集成成本极低（未来下游策略主题使用时约半天），且结论可以 KF 条目形式沉淀到主题 research-status。
