## 1. 实验问题

R2 结论认为下一轮应进入实验线 C：reaccept 深度归一化。

本轮先不改策略代码，只做 fixed ticks 邻域补测，回答：

```text
1. DCE.m2601 的 2~3 ticks 优势是否只是偶然；
2. CZCE.SR601 的 3 ticks 改善是否体现更深重新接受质量要求；
3. SHFE.rb2601 失败是否能通过更深 reaccept 条件改善。
```

## 7. 关键发现

### 7.1 DCE.m2601 的有效区间集中在 2~3 ticks

DCE.m2601 呈现明显平台特征：

```text
1 tick: 交易数增加到 20，但收益下降到 1.1432%，worst_net_pct 扩大到 -1.633%；
2 ticks: 收益最高 3.5168%，win_loss_ratio 4.23；
3 ticks: 收益仍较好 2.3928%，win_loss_ratio 4.79，成本压力最低；
4 ticks: 收益接近归零，worst_net_pct 扩大到 -1.893%。
```

这说明 1 tick 太浅，容易把贴边噪声纳入；4 ticks 太深，可能已经错过回归到 POC 的主要空间，或只留下质量不稳定的晚确认样本。

因此，DCE.m2601 的 fixed ticks 证据支持：

```text
2~3 ticks 是当前有效邻域，不是单点偶然。
```

### 7.2 CZCE.SR601 更偏 3 ticks，但不是强主线

SR 的 1 / 2 ticks 都能保持小幅正收益，但成本压力很高：

```text
1 tick cost / avg_net_win = 3.950；
2 ticks cost / avg_net_win = 3.582。
```

3 ticks 明显改善：

```text
net_pnl = 882.8；
win_loss_ratio = 1.95；
worst_net_pct = -0.486%；
cost / avg_net_win = 1.603。
```

4 ticks 虽然成本压力继续下降，但收益转负，take_profit 为 0，说明过深确认后 POC 兑现能力不足。

因此，SR 的结论是：

```text
3 ticks 是当前 fixed ticks 下相对合理的观察点；
但成本安全边际仍不如 DCE.m，不能升为主线。
```

### 7.3 SHFE.rb2601 失败不是 reaccept 深度太浅造成的

rb 从 1 到 4 ticks 全部为负：

```text
1 tick: -4.9460%；
2 ticks: -3.3142%；
3 ticks: -1.8014%；
4 ticks: -0.9011%。
```

加深 reaccept 能减少交易、降低亏损幅度和成本，但没有把结构变成正期望。其问题更像是：

```text
价值区重新接受到 POC 的方向 edge 不成立，
或 POC 目标 / 持仓时间 / 成本结构与 rb 的日内行为不匹配。
```

rb 继续保留为负面对照即可，不应纳入主线优化。

## 8. 本轮结论

```text
fixed ticks 邻域验证支持 R2 的品种分层判断。
```

具体结论：

```text
DCE.m2601：2~3 ticks 构成有效邻域，当前主线保留；
CZCE.SR601：3 ticks 相对更好，但成本安全边际仍薄，仅保留观察；
SHFE.rb2601：加深 reaccept 只能减亏，不能扭转负期望，继续排除为主线。
```

本轮没有证明 fixed ticks 是最终表达方式，只证明当前 fixed ticks 下：

```text
过浅会增加噪声和成本；
过深会损失 POC 空间或样本质量；
有效区域大致集中在 2~3 ticks，但不同品种偏好不同。
```

## 9. 下一轮建议

下一轮应进入真正的 reaccept 深度归一化诊断，而不是继续扩大 fixed ticks：

```text
1. 为每笔入场记录 previous_value_area_width；
2. 计算 reaccept_depth / previous_value_area_width；
3. 如可用，计算 reaccept_depth / ATR 或近 N 根 5m 平均波动；
4. 计算 entry 在前日 value area 内部的位置分位；
5. 对 DCE.m2601 与 CZCE.SR601 比较有效样本是否落在相似归一化区间。
```

如果归一化后 DCE.m 与 SR 的有效区间能统一，才值得继续做参数平台验证；否则应承认该结构需要品种分层，而不是强行寻找统一参数。
