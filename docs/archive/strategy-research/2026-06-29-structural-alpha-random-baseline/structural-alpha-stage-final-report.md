## 1. 阶段问题

本阶段从原始问题：

```text
共识价格区间附近的接受 / 拒绝、假突破、重新接受或压力释放，
是否能形成比普通指标边界更好的结构型 alpha 实验场？
```

进一步补充了随机对照问题：

```text
这些结构入口是否至少比匹配随机开仓更有信息量？
```

本阶段不以找到可上线策略为目标，而是判断：

```text
结构入口是否彻底等同随机；
若不等同随机，优势来自方向层、事件层、风险空间层、退出层还是状态标签。
```

## 3. 总体结论

本阶段的核心结论是：

```text
结构型入口路线不是无效；
但不同结构的价值层级发生分化。
```

更具体：

```text
方向层：部分结构明显优于随机方向；
风险空间层：原始入场 / strict failure / 止损塑形普遍不足；
接受质量层：价值区重新接受深度能显著改善风险空间；
退出层：退出规则会显著影响方向 edge 的兑现，但本阶段不继续深耕；
状态标签层：成交量冲击等变量更适合降级为质量标签，而不是独立入口。
```

因此，本阶段不是“所有结构入口失败”，而是：

```text
已确认部分结构存在方向信息，
下一阶段应选择方向，而不是继续证明这些结构是否有价值。
```

## 4. 结构路线结论矩阵

| 结构路线 | 随机对照结论 | 当前定位 |
|----------|--------------|----------|
| 前日价值区 VAH / VAL 重新接受 | 随机方向基准显著优于；原始风险空间一般；深度重新接受后风险空间显著改善 | 最强主线，保留 |
| 前日高低点重新接受 | 有弱方向信息，但同方向风险空间优势不足 | 弱线索，非优先 |
| 成交量爆发边界 | 胜率 / 方向信息强，但赔率和左尾差 | 降级为状态 / 质量标签 |
| 小时等高 / 等低流动性扫单 | 当前方向显著差于随机方向 | 当前入口暂停，可反向诊断 |
| 低波收敛再启动 | 当前参数下 0 交易，无法评价 | 交易机会不足，暂停 |
| Initial Balance 重新接受 | 历史报告存在，但策略实现未保留，无法同口径复跑 | 报告级历史实验，不纳入当前随机对照结论 |
| 前日边界 + 成交量过滤 | 有策略实现，但批量随机对照过慢未完成 | 非核心缺口，后续可单独补 |
| 滚动日内价值区 | 更像状态标签，不是标准入口 | 后续若做，应采用分桶随机对照 |

## 5. 最重要的正反馈

价值区 VAH / VAL 重新接受是本阶段最明确的保留方向。

原始版本结论：

```text
价值区方向假设有信息；
但重新接受 K 收盘 + 假突破极值 strict failure 的原始风险空间，
没有显著优于同事件同方向随机风险空间。
```

深耕后结论升级为：

```text
等待价格重新接受到边界内侧 2~3 ticks，
能显著改善价值区方向 edge 的风险空间转化。
```

其中 DCE.m2601 的 `min_reaccept_ticks = 2` 改善最明确：

```text
平均 MAE 下降；
平均 MFE 上升；
最大单笔亏损下降；
亏损簇变浅；
成本 / 平均盈利下降；
相对同方向随机达到高分位。
```

SR601 也支持重新接受深度变量有效，但更偏向品种差异：

```text
2 ticks 更偏胜率优势；
3 ticks 对尾部风险更友好。
```

## 6. 当前不继续深耕的线索

### 6.1 退出策略

本阶段确认：

```text
退出层会显著影响方向 edge 的兑现质量。
```

但当前不继续展开止盈 / 时间退出优化。已保留后续方向：

```text
partial_target_ratio = 0.3 / 0.5 / 0.7 of POC distance
take_profit_r = 0.5 / 0.75 / 1.0
MFE 回撤退出
time_exit 前小 MFE 主动兑现
```

暂停原因：

```text
当前阶段目标是判断结构入口和风险空间是否有价值，
不应过早转入退出参数优化。
```

### 6.2 成交量质量标签

成交量爆发边界方向 / 胜率信息明显，但赔率过差。当前不作为独立入口继续优化，保留为：

```text
边界质量标签；
假突破冲击强度诊断；
后续结构过滤变量。
```

### 6.3 未完成覆盖项

未完成项不改变阶段结论：

```text
IB 无可复跑实现；
prevday_volume_filter 性能过慢；
rolling intraday value area 更适合作为状态标签；
500/1000 seeds 规模留待正式验证。
```

这些属于覆盖完整性问题，不影响当前判断：

```text
结构入口并非整体等同随机，
但下一阶段应选择主线，而不是继续铺满所有随机对照。
```

## 7. 对原短期计划的回答

原短期计划希望验证：

```text
共识边界 + 接受 / 拒绝 + 明确失败边界 + 盈利上界 + 账户风险预算
是否能形成成本后正期望结构。
```

本阶段回答如下：

| 维度 | 阶段回答 |
|------|----------|
| 共识边界是否有意义 | 有，至少价值区、前日边界、成交量冲击不等同随机 |
| 方向假设是否有信息 | 部分有，价值区和成交量最明显 |
| 原始 strict failure 是否足够 | 普遍不足，价值区原始风险空间未显著优于同方向随机 |
| 接受质量是否重要 | 重要，价值区重新接受深度是关键改善变量 |
| 盈利上界是否充分 | POC 有价值，但对部分交易偏远 |
| 时间退出是否简单放宽 | 不应简单放宽，18/24 bars 放大左尾 |
| 是否已可进入上线验证 | 否 |
| 是否值得继续研究 | 是，但应聚焦价值区主线和结构塑形，不再广撒入口 |

## 8. 阶段决策

本阶段结题决策：

```text
结束“共识边界直接入口 + 随机对照铺面”阶段。
```

下一阶段不建议继续问：

```text
还有哪个入口没试？
```

而应问：

```text
价值区方向 edge 如何在账户风险预算、失败边界质量、成本空间和品种适配下稳定兑现？
```

推荐下一阶段主线：

```text
价值区 VAH / VAL 重新接受
+ POC 空间
+ price_raw_rr 预筛
+ min_reaccept_ticks 2~3
+ max_hold_bars 约 12
→ 账户风险预算 / 品种适配 / 尾部风险补证
```

## 9. 后续建议

后续优先级：

1. **账户风险预算补证**：合约乘数、最小手数、实际仓位、2%~3% 风险是否可执行；
2. **品种适配**：DCE.m 与 SR 的最优机制不同，需建立品种性格解释；
3. **波动归一**：`2~3 ticks` 不能直接视为最终参数，应测试 ATR、价值区宽度百分比或价值区内侧分位；
4. **尾部风险**：最大单亏、亏损簇、force_flat / stop_loss 左尾是否可控；
5. **退出策略**：已确认有价值，但暂缓到后续阶段；
6. **成交量标签**：不作为入口，但可作为质量过滤候选。
