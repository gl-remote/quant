## 1. 阶段问题

本阶段不再问：

```text
结构入口能不能直接转正？
```

而是问：

```text
结构入口是否至少比匹配随机开仓更稳定、更收敛？
```

如果结构与随机无差异，则暂停；如果显著优于随机但仍未转正，则保留为方向线索、诊断变量或风险空间补证对象。

## 4. 阶段结论

### 4.1 不是所有结构都等同随机

价值区、前日高低点、成交量爆发边界都显示出一定方向 / 胜率信息。

尤其：

```text
价值区：随机方向胜率约 34%~36%，原结构 44.44%；
成交量爆发：随机方向胜率约 56.79%，原结构 83.33%。
```

因此不能说这些结构“彻底失去意义”。

### 4.2 但风险空间层普遍未通过

目前没有一个结构明确通过：

```text
同事件同方向随机风险空间对照
```

这说明：

```text
事件 / 方向可能有信息，
但当前入场价格、strict failure、止损和账户风险空间塑形没有证明有效。
```

这比“入口结构整体失败”更精确。

### 4.3 高胜率结构的问题不是方向，而是赔率和左尾

成交量爆发边界胜率很高，但：

```text
win_loss_ratio = 0.2271
```

说明其高胜率被极低盈亏比和成本吞噬。它更适合作为质量标签，而不是直接入口。

### 4.4 小时流动性扫单方向假设应暂停

小时流动性扫单的随机方向结果显著优于原方向：

```text
原结构收益分位仅 20%；
原结构胜率比随机方向低约 15~16 个百分点。
```

当前方向假设不成立。

## 5. 阶段决策

| 结构 | 决策 |
|------|------|
| 价值区 VAH / VAL 重新接受 | 保留为最强方向线索；不急于优化，先完成剩余随机对照 |
| 前日高低点重新接受 | 保留为弱方向线索；非优先 |
| 成交量爆发边界 | 降级为状态 / 质量标签 |
| 小时流动性扫单 | 暂停入口；可作为反向诊断候选 |
| 低波收敛再启动 | 交易机会不足，暂停 |
| 前日边界 + 成交量过滤 | 待单独跑完后再定 |

## 6. 下一步

优先不是继续优化单个结构，而是把随机对照素材耗尽：

```text
1. 单独处理 prevday_volume_filter；
2. 确认 IB 是否有可复跑策略实现；
3. 若无 IB 代码，标记为报告级历史实验，无法进入同口径随机对照；
4. 对滚动价值区标签另开“分桶随机对照”，而不是入口随机对照；
5. 然后更新 roadmap：当前阶段结论是“方向层部分有效，风险空间层未通过”。
```
