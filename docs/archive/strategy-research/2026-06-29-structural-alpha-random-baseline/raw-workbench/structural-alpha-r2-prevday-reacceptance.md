## 1. 核心问题

### 1.1 研究边界说明

本轮研究对象是**跨天共识边界**，不是跨天持仓策略。

更准确的定义是：

```text
前日高 / 前日低 / 昨收等跨天边界
+ 当日日内价格对该边界的假突破、重新接受或拒绝
+ 日内执行、日内退出
```

因此，本文当前所有 `1m` / `5m` 对照只是在比较：

```text
用多粗的日内 K 线确认“重新接受 / 拒绝”更少噪声、更少交易成本
```

它不能回答：

```text
前日高低点重新接受后，隔夜或次日是否存在更大的跨天盈利上界
```

当前策略不包含：

```text
隔夜持仓
T+1 开盘 / 收盘退出
多日时间退出
次日 VWAP / 次日收盘盈利上界
隔夜跳空风险诊断
```

所以本轮应称为：

```text
跨天共识边界的日内重新接受执行实验
```

而不是：

```text
前日高低点的跨天 alpha 持仓研究
```

若后续要做真正的跨天研究，需要另起一轮，重新定义持仓周期、盈利上界、失败边界、隔夜风险和账户风险预算。

### 1.2 本轮核心问题

本轮实验接在 IB 重新接受方向之后，优先验证更高一级、盈利上界可能更大的共识边界：

```text
前日高 / 前日低
→ 边界外假突破后重新接受 / 拒绝
→ 是否比 IB 边界提供更大的价格原始盈亏比和账户原始盈亏比
→ 是否能覆盖手续费和滑点
→ 是否具备跨品种稳定性
```

本轮不做：

```text
不做 Optuna
不做 Walk-Forward
不叠加 MACD / KDJ / 均线确认
不引入 VAH / VAL / POC
不为了交易数放宽前日高低点定义
```

## 2. 实验定义

| 项目 | 定义 |
| --- | --- |
| 实验版本 | `structural-alpha-r2` |
| 策略代号 | `prevday_reacceptance` |
| 候选共识价格区间 | 前一交易日高点 / 低点 |
| 结构来源 | Price Action / 日内流动性扫单 / Auction |
| 传统解释 | 前日极值被突破后无法维持，边界外追单失败，价格重新接受昨日区间 |
| 结构塑形解释 | 前日高低点是更高共识边界；假突破极值提供严格失败边界；昨收 / 当日开盘 / 前日区间中轴提供短期盈利上界 |
| 方向假设 | 上破前日高后重新收回前日高下方做空；下破前日低后重新收回前日低上方做多 |
| 严格失败边界 | 假突破极值外加 `failure_buffer_ticks` |
| 严格失败距离 | 入场价到严格失败边界的距离 |
| 目标止盈 | 昨收 / 当日开盘 / 前日区间中轴 / 对侧边界 / R 倍数 |
| 时间退出 | 入场后 `max_hold_bars` 或日内 `force_flat_time` |
| 入场方式 | 边界外触发突破状态后，收盘价重新回到昨日区间内并满足最小突破幅度 |
| 退出方式 | 严格失败边界、主动止盈、时间退出、日内强平 |

### 2.1 多头结构

```text
已知前日 High / Low / Close
→ 价格向下跌破前日 Low 至少 min_breakout_ticks
→ 价格收盘重新回到前日 Low 上方
→ 做多
→ 严格失败边界 = 假突破低点 - failure_buffer_ticks
→ 盈利上界 = 昨收 / 当日开盘 / 前日区间中轴 / 前日 High / R 倍数
```

### 2.2 空头结构

```text
已知前日 High / Low / Close
→ 价格向上突破前日 High 至少 min_breakout_ticks
→ 价格收盘重新回到前日 High 下方
→ 做空
→ 严格失败边界 = 假突破高点 + failure_buffer_ticks
→ 盈利上界 = 昨收 / 当日开盘 / 前日区间中轴 / 前日 Low / R 倍数
```

观察：

1. 前日高低点重新接受比 IB 方向明显改善，尤其 5m + 昨收目标组把亏损压到 `-1.91%`。
2. 改善主要来自更粗周期、减少交易、限制每日交易和 1.5x 止损放宽，而不是原始结构直接转正。
3. 最佳组成本后仍为负；总成本 `3,771.77` 大于最终亏损绝对值，说明毛收益可能接近正，但安全边际不足。
4. 平均盈利 `499` 小于平均亏损 `733`，胜率虽高，但盈亏比仍弱。
5. 2R 目标没有改善，说明 MFE 不足以稳定支持更远盈利上界。

## 5. 关键诊断

| 诊断项 | 当前结果 | 含义 |
| --- | --- | --- |
| 候选区间是否客观 | 通过 | 前日高低点可由历史 K 线客观定义 |
| 严格失败边界是否清晰 | 通过 | 假突破极值外加 buffer，可映射严格失败边界 |
| 价格原始盈亏比 | 未通过 | 2R 目标不改善，昨收目标更稳但平均盈利仍小于平均亏损 |
| 账户原始盈亏比 | 未通过 | 2% 风险预算下净收益仍为负，安全边际不足 |
| 接受 / 拒绝质量 | 偏弱 | 5m 过滤和每日 1 笔能改善胜率，但更多像降频过滤 |
| 成本空间 | 未通过 | 最佳组总成本 `3,771.77`，成本后仍 `-1,911.77` |
| 跨品种一致性 | 未通过 | 最佳结构迁移到 4 个品种全部为负 |

失败层级：

```text
共识边界和严格失败边界成立
→ 但重新接受后的 MFE 不足
→ 更远目标无法兑现
→ 近目标胜率提升但平均盈利偏小
→ 成本后期望仍为负
→ 跨品种不稳定
```

## 6. 临时结论

当前判断：

```text
前日高低点假突破重新接受方向明显优于 IB 重新接受，但仍未通过结构型 alpha 标准。
```

主要证据：

1. 最佳组 `316` 已接近盈亏平衡，但仍为 `-1.91%`。
2. 改善来自 5m、止损放宽、每日 1 笔和昨收目标的组合，不足以说明原始结构成立。
3. 跨品种确认全部为负。
4. 不进入 Optuna / Walk-Forward。
