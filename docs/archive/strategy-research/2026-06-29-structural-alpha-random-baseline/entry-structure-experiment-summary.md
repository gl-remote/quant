> 状态：已归档

## 1. 阶段背景

本摘要压缩 `docs/workbench` 中结构型 alpha 第一批入口实验，覆盖：

```text
IB 重新接受；
前日高低点重新接受；
价值区 VAH / VAL 重新接受；
成交量爆发边界；
前日边界 + 成交量过滤；
价值区 POC 路径；
小时等高 / 等低流动性扫单；
低波收敛再启动；
持续 / 滚动价值区上下文。
```

这些实验的共同目标是判断：

```text
共识价格区间是否能直接作为结构型 alpha 入口。
```

## 2. 入口结构摘要

| 结构 | 初始实验结论 | 后续定位 |
|------|--------------|----------|
| Initial Balance 重新接受 | 成本后期望不足，严格失败和成本空间不支持 | 历史实验；当前实现未保留 |
| 前日高低点重新接受 | 优于 IB，但仍未转正，成本和跨品种稳定性不足 | 弱方向线索 |
| 价值区 VAH / VAL 重新接受 | 全样本未通过，但 POC 空间 + price_raw_rr 子样本出现线索 | 主线保留 |
| 成交量爆发边界 | 作为主边界未通过 | 降级为质量标签 |
| 前日边界 + 成交量过滤 | breakout shock 有信息，但不适合作硬入口 | 质量标签候选 |
| 价值区 POC 路径 | 简单路径早退非通用有效 | 路径质量可作为诊断变量 |
| 小时流动性扫单 | 当前方向假设失败 | 暂停 / 可反向诊断 |
| 低波收敛再启动 | 交易机会不足或结构不成立 | 暂停 |
| 持续价值区 / 滚动日内价值区 | 不支持“越持续越强”的简单命题，但可表达上下文差异 | 状态标签 |

## 3. 主要经验

### 3.1 低验证成本不是优势本身

很多结构能定义清晰失败边界，但仍无法通过：

```text
盈利上界不足；
成本空间不足；
严格失败边界容易吞掉多笔小盈利；
原始账户盈亏比不足；
跨品种不稳定。
```

### 3.2 直接入口大多未通过

多数共识边界直接做入口时，无法形成稳定成本后正期望。

但这不等于结构完全无效，而是说明：

```text
共识边界本身可能提供方向 / 状态信息；
原始入场、失败边界、止损、止盈和账户风险塑形不足。
```

### 3.3 价值区方向出现最强线索

价值区 VAH / VAL 重新接受是唯一从失败实验中继续收敛出正反馈的结构：

```text
入场到 POC 有足够空间；
price_raw_rr 预筛改善结果；
DCE.m 与 SR 存在正向样本；
rb 等品种价值区结构质量差，不能一刀切。
```

### 3.4 成交量更适合作为质量标签

成交量爆发边界作为主入口未通过，但后续随机对照显示其方向 / 胜率信息较强。当前定位：

```text
边界冲击质量标签；
假突破事件强度变量；
后续辅助过滤候选。
```

## 4. 原始入口阶段结论

本阶段原始入口实验的结论不是：

```text
所有结构入口都失败，没有研究价值。
```

而是：

```text
共识边界直接入口普遍不足；
需要通过随机对照判断事件 / 方向是否仍有信息；
若有信息，再进入风险空间、接受质量、账户预算和退出层塑形。
```
