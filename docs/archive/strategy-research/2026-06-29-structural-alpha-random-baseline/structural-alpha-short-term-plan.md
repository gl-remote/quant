## 1. 阶段目标

本阶段目标是把长期框架中的结构型 alpha 拆成可验证的短期实验：

```text
共识价格区间
→ 明确失败边界
→ 可估算的短期盈利上界
→ 可执行的 2%~3% 单次账户风险
→ 足够的价格原始盈亏比和账户原始盈亏比
→ 可评估的胜率 / 盈亏比转化效率
→ 成本后长期正期望和安全边际
```

核心问题：

```text
共识价格区间附近的接受 / 拒绝、假突破、重新接受或压力释放，
是否能形成比普通指标边界更好的结构型 alpha 实验场？
```

## 2. 候选区间

阶段优先候选：

| 候选区间 | 主要验证问题 |
|----------|--------------|
| Initial Balance 高 / 低点 | 日内早盘共识边界是否能提供清晰接受 / 拒绝、假突破和重新接受结构 |
| 前日高 / 低点、昨收、开盘区间边界 | 高频可观察边界附近的突破失败或重新接受是否优于普通前 N 高低点 |
| VAH / VAL / POC 等价值区边缘 | 市场是否接受更高 / 更低价值区，价值区边缘是否能定义失败边界和盈利上界 |
| 支撑 / 阻力 / 密集成交区边缘 | 密集成交区边缘是否能提供更强接受 / 拒绝质量，降低快速反向 MAE |
| 假突破后的重新接受 / 拒绝 | 假突破极值、重新收回边界、边界外无法维持是否能形成明确失败边界 |

本阶段不优先普通指标边界、均线确认、MACD / KDJ 叠加和大规模参数搜索。

## 3. 评价顺序

每轮实验按以下顺序判断：

```text
是否符合有限资金约束
→ 单次账户风险是否能控制在 2%~3%
→ 是否存在明确共识价格区间
→ 严格失败边界是否清楚
→ 预期盈利上界是否可定义
→ 价格原始盈亏比是否足够
→ 账户原始盈亏比是否足够
→ 严格失败边界快速再触及率是否可接受
→ 接受 / 拒绝质量是否足够
→ 主动止盈、时间退出或有限止损放宽是否改善转化效率
→ 成本后期望是否为正
→ 正期望是否有足够安全边际
→ 尾部亏损和超预算亏损是否受控
→ 交易机会是否足够评价
→ 参数邻域是否存在稳定平台
```

收益率、夏普、最大回撤和最优参数只作为后置参考，不能替代结构诊断。

## 5. 阶段执行结果

本阶段实际完成：

```text
IB 重新接受历史实验；
前日高低点重新接受；
价值区 VAH / VAL 重新接受；
成交量爆发边界；
前日边界 + 成交量过滤；
小时流动性扫单；
低波收敛再启动；
持续 / 滚动价值区上下文诊断；
结构入口随机对照补充；
价值区深度重新接受机制深耕。
```

阶段结论：

```text
共识边界直接入口并未形成可上线策略；
但部分结构入口显著不同于随机，尤其价值区和成交量冲击；
价值区 VAH / VAL 重新接受成为最强保留主线；
原始入场 / strict failure / 风险空间普遍不足，需要结构塑形；
退出策略和成交量标签有后续价值，但本阶段不继续深挖。
```

## 6. 阶段暂停 / 保留决策

| 结构 | 决策 |
|------|------|
| 价值区 VAH / VAL 重新接受 | 保留，进入下一阶段主线候选 |
| 前日高低点重新接受 | 弱线索，非优先 |
| 成交量爆发边界 | 降级为状态 / 质量标签 |
| 小时等高 / 等低流动性扫单 | 当前方向暂停，可作为反向诊断 |
| 低波收敛再启动 | 交易机会不足，暂停 |
| IB 重新接受 | 历史实现未保留，不进入同口径随机对照 |
| 滚动价值区 | 状态标签，若继续应做分桶随机对照 |

## 7. 后续方向

本阶段结束后，不建议继续广撒新入口。

建议下一阶段聚焦：

```text
价值区 VAH / VAL 重新接受
+ POC 空间
+ price_raw_rr 预筛
+ min_reaccept_ticks 2~3
+ max_hold_bars 约 12
→ 账户风险预算 / 品种适配 / 尾部风险补证
```

需要补证：

```text
2%~3% 单次账户风险是否可执行；
合约乘数、最小手数、滑点和跳空后的实际风险；
DCE.m 与 SR 的品种机制差异；
2~3 ticks 是否可波动归一；
最大单亏、亏损簇、force_flat / stop_loss 左尾；
成交量冲击是否能作为质量标签。
```
