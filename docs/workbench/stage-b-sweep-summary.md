### C2 = 0 trades 全 cell (不是 bug，是 spec 5.2 X_s 极值化的必然结果)

- C2 判定：`A_s >= 1 且 B_s(t) < B_s^-`（严格小于）
- Attempt event 更新时：`B_s^- := B_s(t)`，即上一次事件的 B_s
- `X_s(t) = max{H_i}` 是最近 n_step 内的极值，后续 attempt 的 breakout 极值只可能等于旧极值（同 bar）或更远，不会更弱
- 除非把 `X_s` 从"极值"降级为"最近一次突破 bar 的 high"（非累积极值化），否则 C2 恒为假
- 结论：C2 目前是形式条件，spec 5.2 决定其永远不触发；如果要让 C2 有意义需要修 spec

### C3 独立成分具有实质经济意义

- 定义：`A_s >= 1 且 Z_s^- = False`（次次尝试且相邻两次事件间未触碰 POC）
- 与 C1（首次尝试）完全独立事件集
- n_profile=144 (12h) 档在 Group_P 4/6 有 trade 合约上正、ret_mean +1.10、rgr 0.909
- 相邻 attempt 间"未触 POC"的语义在较长 profile 窗口下可分辨 (12h > 8h > 4h)

### 双 Q 判据判定

- Q_return (Group_P 均值提升)：C3 @ n=144 达标 (ret_mean +1.10 vs baseline 0)
- Q_generalize (Group_M >=5/8 profitable)：不达标 (只 2/8 profitable，5/8 无 trade)
- 结论：按 experiment-plan 7.1 -> 走 feature-only 降级路径
