## 7. 各方向解读（为何 fail / 回退）

### C.1 品种筛选（S2 vs B0）
- ΔSharpe = **-0.27**，p_boot = **0.0000**（cluster bootstrap 100% 次 S2 更差）→ 双判据均输。
- 解读：§4.1「按品种类型筛选 tier」剔除了原本盈利的 tier/品种组合（A 类禁用 L_seg3/S_seg34、B 类禁用 L_seg12/S_seg12 后，组合反而变差）。
- **结论：回退 S1（全品种 5 档）**，不启用品种筛选。

### C.2 信号强度加权（W1/W2/W3 vs W0）
- W1 持平 B0（ΔSh=-0.01，ann 12.50%）· W2 最差（ΔSh=-0.32，ann 5.65%）· **W3（spec 默认）也差（ΔSh=-0.22，ann 8.51%）**。
- 诊断：w_strength 与单笔 pnl_net_bps 相关性 ≈ 0（W1 corr=+0.06，W3 corr=**-0.07**）。即 spec 定义的强度信号与收益几乎无关，W3 甚至给「高权重信号」更低收益 → 加权拖累。
- **结论：回退 W0（等权）**，强度加权在本数据上无效且方向不明。

### C.3 多空权重（VW1/VW2 vs VW0）
- VW1 持平 B0（ΔSh=-0.01）· VW2 拖累（ΔSh=-0.23，ann 11.58%）。
- 解读：IR 比例 / 频率平衡均无法在「多空等权」基础上增配出更高夏普（本主题多空贡献度本就均衡，L/S ≈ 50%/50%）。
- **结论：回退 VW0（多空等权）**。
