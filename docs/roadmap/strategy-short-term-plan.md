## 1. 短期定位

短期研究不以原创预测型 alpha discovery 为主线，也不以长期 beta / 风险溢价暴露为主线。

当前阶段的目标是把长期框架中的结构型 alpha 拆成可验证的短期实验：

```text
共识价格区间
→ 明确失败边界
→ 可估算的短期盈利上界
→ 可执行的 2%~3% 单次账户风险
→ 足够的价格原始盈亏比和账户原始盈亏比
→ 可评估的胜率 / 盈亏比转化效率
→ 成本后长期正期望和安全边际
```

因此，短期研究的核心不是寻找更准的入场信号，而是验证：是否存在一类市场共同关注的价格区间，能在小资金账户约束下提供可重复、可解释、可风控的结构性优势。

## 2. 当前阶段核心问题

当前阶段优先回答一个问题：

```text
共识价格区间附近的接受 / 拒绝、假突破、重新接受或压力释放，
是否能形成比普通指标边界更好的结构型 alpha 实验场？
```

这个问题需要拆成以下顺序验证：

```text
是否存在明确的市场共识边界
→ 是否能定义方向假设
→ 是否能定义严格失败边界
→ 严格失败边界能否映射到 2%~3% 单次账户风险
→ 是否能定义短期盈利上界
→ 价格原始盈亏比是否足够
→ 账户原始盈亏比是否足够
→ 接受 / 拒绝质量是否足够
→ 主动止盈、时间退出或有限止损放宽是否改善胜率 / 盈亏比转化效率
→ 成本后期望和安全边际是否成立
```

如果前置问题不能成立，不进入参数搜索、Walk-Forward 或复杂过滤叠加。

## 3. 短期实验哲学

短期实验采用“结构型 Alpha 验证”作为实验哲学。

| 组成 | 定位 |
|------|------|
| 共识价格区间 | 研究对象，不是预测信号 |
| 方向假设 | 用于定义本轮方向选择，不假设能稳定预测未来 |
| 严格失败边界 | 理论上证明方向假设失败的位置 |
| 短期盈利上界 | 单次方向判断在有限持仓周期内可捕获的空间估计 |
| 价格原始盈亏比 | 盈利上界 / 严格失败距离，判断价格结构是否有空间 |
| 单次账户风险 | 严格失败触发时账户损失，目标控制在 2%~3% |
| 账户原始盈亏比 | 预期账户盈利 / 单次账户风险，判断资金口径是否成立 |
| 接受 / 拒绝质量 | 判断边界是否有真实市场行为，而不是机械触碰指标线 |
| 主动止盈 / 时间退出 | 在原始盈亏比足够时，将部分盈亏比转化为胜率 |
| 有限止损放宽 | 诊断是否能吸收正常噪声；放宽后必须同步降仓 |
| 成本后期望 | 判断结构是否具备可交易的正期望和安全边际 |

核心判断不是：

```text
这个信号是不是更准？
```

而是：

```text
这个共识价格区间是否更容易低成本判断方向选择失败？
失败边界和盈利上界是否清楚？
在 2%~3% 账户风险预算下，价格原始盈亏比和账户原始盈亏比是否足够？
胜率 / 盈亏比转化后，成本后期望是否仍为正？
```

## 4. 当前实验命名

下一阶段优先实验命名为：

```text
structural-alpha-r1
```

含义：

```text
从历史 low-validation-cost-r1/r2 的“低验证成本”检查，
升级为“共识边界 + 盈利上界 + 账户风险预算 + 胜率 / 盈亏比转化”的完整结构型 Alpha 验证。
```

`low-validation-cost` 仍保留为诊断维度，但不再作为唯一实验目标。仅有低验证成本，不足以说明结构有效。

## 5. 候选区间来源

### 5.1 优先候选

短期优先研究更可能被市场共同观察的边界，而不是普通指标边界。

| 候选区间 | 主要验证问题 |
|----------|--------------|
| Initial Balance 高 / 低点 | 日内早盘共识边界是否能提供清晰接受 / 拒绝、假突破和重新接受结构 |
| 前日高 / 低点、昨收、开盘区间边界 | 高频可观察边界附近的突破失败或重新接受是否优于普通前 N 高低点 |
| VAH / VAL / POC 等价值区边缘 | 市场是否接受更高 / 更低价值区，价值区边缘是否能定义失败边界和盈利上界 |
| 支撑 / 阻力 / 密集成交区边缘 | 密集成交区边缘是否能提供更强接受 / 拒绝质量，降低快速反向 MAE |
| 假突破后的重新接受 / 拒绝 | 假突破极值、重新收回边界、边界外无法维持是否能形成明确失败边界 |

每轮只选择一个最简单的候选区间，不同时堆叠多个体系。

### 5.2 暂不优先方向

| 候选方向 | 暂不优先原因 |
|----------|--------------|
| 普通布林带收缩后直接突破 | 历史实验已显示容易出现成本空间不足和接受质量不足 |
| 布林带突破后回踩边界再启动 | 样本减少且止损放宽不稳定改善，不继续围绕同一入口微调 |
| 普通前 N 高 / 前 N 低突破后首次回踩 | 低验证成本成立但严格边界快速再触及率过高，需要升级为更高共识边界 |
| 单纯均线趋势回踩 | 可作为趋势背景诊断，但不能单独作为市场逻辑 |
| MACD / KDJ / 均线确认叠加 | 容易重新滑向预测型信号组合，不作为结构型 alpha 来源 |

## 6. 结构塑形翻译要求

引用任何成熟体系或指标前，必须完成结构塑形翻译：

```text
传统交易理由是什么
→ 它对应哪个共识价格区间
→ 方向假设是什么
→ 严格失败边界在哪里
→ 入场到失败边界的价格验证成本是多少
→ 严格失败边界能否映射到 2%~3% 单次账户风险
→ 预期盈利上界在哪里
→ 价格原始盈亏比是否足够
→ 账户原始盈亏比是否足够
→ 接受 / 拒绝证据是什么
→ 是否需要主动止盈、时间退出或有限止损放宽
→ 转化后成本后期望是否为正，安全边际是否足够
```

如果一个体系只能说明“这里可能涨 / 跌”，但不能帮助定义共识边界、失败边界、盈利上界、账户风险预算和成本后期望，则不进入短期主线。

## 7. 每轮实验结构

### 7.1 回测前预筛

每轮实验在实现或回测前先检查：

| 项目 | 要求 |
|------|------|
| 候选区间 | 可以客观定义，不依赖事后主观画线 |
| 方向假设 | 多空方向和失效条件清楚 |
| 严格失败边界 | 理论失效位置明确 |
| 严格失败距离 | 不应近到被正常噪声反复触及，也不应远到无法执行 2%~3% 账户风险 |
| 短期盈利上界 | 入场前可以估算，不能只依赖无限趋势展开 |
| 价格原始盈亏比 | 盈利上界 / 严格失败距离应有足够空间 |
| 账户风险预算 | 合约乘数、最小手数、滑点、跳空后仍能控制单次账户风险 |
| 成本空间 | 预期平均账户盈利应显著大于手续费和滑点 |

预筛不通过时，不进入回测调参。

### 7.2 最小实验对照

每轮至少比较以下结构：

| 设置 | 目的 |
|------|------|
| 严格失败边界退出 | 观察原始结构是否具备低验证成本和足够原始盈亏比 |
| 主动止盈 / 时间退出 | 观察是否能在盈利上界附近提高兑现质量 |
| 有限止损放宽 + 同步降仓 | 诊断吸收正常噪声后，胜率提升是否覆盖盈亏比和仓位下降成本 |

重点不是寻找最优参数，而是判断收益分布改善来自哪一层结构。

### 7.3 暂不处理

当前阶段不做：

- 大规模 Optuna / Bayesian search；
- Walk-Forward；
- 仓位优化；
- 多体系复杂叠加；
- 为凑交易数放宽区间定义；
- 为提高胜率取消止损或无限放宽止损；
- 亏损后加仓摊低；
- 根据单次收益结果频繁切换候选区间。

## 8. 评价顺序

每轮实验按以下顺序判断：

```text
是否符合有限资金约束
→ 单次账户风险是否能控制在 2%~3%
→ 是否存在明确共识价格区间
→ 严格失败边界是否清楚
→ 预期盈利上界是否可定义
→ 价格原始盈亏比是否足够
→ 账户原始盈亏比是否足够
→ 严格失败边界快速再触及率是否可接受
→ 接受 / 拒绝质量是否足够
→ 主动止盈、时间退出或有限止损放宽是否改善转化效率
→ 成本后期望是否为正
→ 正期望是否有足够安全边际
→ 尾部亏损和超预算亏损是否受控
→ 交易机会是否足够评价
→ 参数邻域是否存在稳定平台
```

收益率、夏普、最大回撤和最优参数只作为后置参考，不能替代结构诊断。

## 9. 关键观察指标

每轮实验记录应至少包含以下字段：

| 字段 | 说明 |
|------|------|
| 实验版本 | 例如 `structural-alpha-r1` |
| 候选共识价格区间 | Initial Balance、前日高低点、价值区边缘等 |
| 结构来源 | Price Action、Wyckoff、Auction / Market Profile 等 |
| 传统解释 | 该体系通常如何解释该位置 |
| 结构塑形解释 | 本项目为什么认为它是结构型 alpha 实验场 |
| 方向假设 | 本轮多空方向选择假设 |
| 严格失败边界 | 理论上方向假设失效的位置 |
| 严格失败距离 | 入场到严格失败边界的距离 |
| 预期盈利上界 | 有限持仓周期内可捕获空间 |
| 价格原始盈亏比 | 预期盈利上界 / 严格失败距离 |
| 单次账户风险 | 严格失败触发时的账户损失，占账户比例 |
| 账户原始盈亏比 | 预期账户盈利 / 单次账户风险 |
| 接受 / 拒绝证据 | 重新接受、失败测试、边界外无法维持等 |
| 严格边界快速再触及率 | 判断边界是否过近、是否只是承受正常噪声 |
| MAE / MFE | 判断接受质量、止损和止盈是否合理 |
| 实际止损距离 | 放宽后真实止损距离 |
| 止损放宽倍数 | 实际止损距离 / 严格失败距离 |
| 仓位调整倍数 | 放宽止损后为维持 2%~3% 风险所需降仓比例 |
| 成本后胜率 | 扣除成本后的胜率 |
| 成本后盈亏比 | 扣除成本后的平均盈利 / 平均亏损 |
| 盈亏平衡胜率 | 根据成本后盈亏比计算的最低胜率 |
| 胜率安全边际 | 实际胜率 - 盈亏平衡胜率 |
| 胜率 / 盈亏比转化效率 | 牺牲多少盈亏比换来多少胜率 |
| 最大单笔亏损 / 平均账户盈利 | 判断一次亏损会回吐多少笔平均盈利 |
| 亏损簇 | 最大连续亏损次数、最差连续亏损金额 |
| 成本 / 平均账户盈利 | 判断小盈利是否被成本吞噬 |
| exit reason 分布 | 止盈、严格失败、实际止损、时间退出、异常退出等 |
| 结论 | 通过、调整边界、调整退出、暂停或更换候选区间 |

## 10. 通过标准

每轮不是要求立即得到可上线策略，而是要求看到结构型 alpha 成立的证据。

| 维度 | 通过标准 |
|------|----------|
| 共识区间 | 区间有明确市场关注基础，可以客观定义 |
| 失败边界 | 严格失败边界清楚，且能映射到 2%~3% 单次账户风险 |
| 盈利上界 | 短期盈利上界可以预先估算 |
| 原始盈亏比 | 价格原始盈亏比和账户原始盈亏比均足够 |
| 接受质量 | 边界附近存在可定义的接受 / 拒绝、假突破或重新接受证据 |
| 边界噪声 | 严格失败边界快速再触及率可接受，不能主要依赖放宽止损硬修 |
| 成本空间 | 平均账户盈利显著大于手续费和滑点 |
| 转化效率 | 主动止盈、时间退出或有限止损放宽改善成本后期望，而不只是提高表面胜率 |
| 账户风险 | 止损放宽后同步降仓，单次账户风险仍不超过 2%~3% |
| 尾部风险 | 最大单笔亏损和亏损簇在小资金账户可承受范围内 |
| 稳定性 | 边界、止盈、止损和时间退出小幅变化不应断崖式退化 |
| 可解释性 | 能说明收益来自哪一层结构，而不是参数偶然性 |

## 11. 暂停条件

出现以下情况时，不继续投入当前结构：

1. 找不到清晰的共识价格区间；
2. 严格失败边界无法定义；
3. 严格失败边界无法映射到 2%~3% 单次账户风险；
4. 预期盈利上界无法定义，或价格原始盈亏比不足；
5. 账户原始盈亏比不足，或平均账户盈利过小；
6. 严格失败边界快速再触及率过高；
7. 接受 / 拒绝证据不足，只是机械触碰指标边界；
8. 合约乘数、最小手数、滑点或跳空导致风险预算不可执行；
9. 主动止盈、时间退出或止损放宽不能改善胜率 / 盈亏比转化效率；
10. 成本后期望无法转正，或安全边际不足；
11. 尾部亏损、超预算亏损或亏损簇不可控；
12. 为维持胜率需要无限放宽止损、加仓摊低或长期扛亏；
13. 收益主要依赖单合约、单时段或尖峰参数；
14. 无法解释收益来自共识价格区间、接受 / 拒绝质量、失败边界、盈利上界、账户风险预算、止损放宽、止盈、时间退出、成本还是市场状态变化。

## 12. 短期执行顺序

```text
启动 structural-alpha-r1
→ 选择一个优先级最高且最容易客观定义的共识价格区间
→ 完成结构塑形翻译
→ 做回测前价格空间和账户风险预筛
→ 实现严格失败边界退出的最小版本
→ 加入主动止盈和时间退出对照
→ 加入有限止损放宽 + 同步降仓对照
→ 输出原始盈亏比、账户原始盈亏比和风险预算诊断
→ 输出接受 / 拒绝质量、MAE / MFE 和严格边界快速再触及率
→ 输出胜率 / 盈亏比转化效率、成本后期望和安全边际
→ 若结构成立，再做边界邻域、多窗口和少量品种一致性检查
→ 若结构不成立，按暂停条件定位失败层级，再决定是否更换候选区间
```
