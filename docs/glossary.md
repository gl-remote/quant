# 研究术语词表

> 类型：Glossary / 项目词表  
> 状态：持续维护  
> 适用范围：策略研究文档、workbench 实验记录、归档复盘  
> 说明：本文件优先解释本项目反复使用的结构术语、自造缩写和诊断标签。成熟交易体系术语只保留本项目中的使用含义，不追求完整教材式定义。

## 1. 使用原则

后续研究文档中，如果出现本文件已定义的术语，优先复用这里的含义。

如果新增临时标签，应满足：

```text
先在实验文档内说明
→ 若多次复用，再补充到本词表
```

术语应尽量服务于本项目的结构塑形框架：

```text
共识价格区间
→ 明确失败边界
→ 短期盈利上界
→ 价格原始盈亏比
→ 账户风险预算
→ 成本后期望
```

## 2. Market Profile / Auction 结构词

### VA / Value Area / 价值区

一段时间内成交最集中的价格区间。项目中通常取覆盖约 70% 成交量或近似成交量的价格范围。

通俗理解：

```text
市场这段时间最认可、交易最密集的价格范围。
```

### VAH / Value Area High

价值区上边界。

项目中用于观察：

```text
价格是否尝试接受更高价值；
上破后又回到 VA 内，是否构成更高价格接受失败。
```

### VAL / Value Area Low

价值区下边界。

项目中用于观察：

```text
价格是否尝试接受更低价值；
跌破后又回到 VA 内，是否构成更低价格接受失败。
```

### POC / Point of Control

一段时间内成交最集中的价格。

项目中主要有两种用法：

1. 作为价值区内部的成交核心；
2. 作为边缘回归交易的短期盈利上界候选。

注意：POC 不是方向预测信号。当前研究结论更倾向于：

```text
POC 是盈利上界 / 状态诊断变量，
不是单独 alpha 来源。
```

### Profile Mode

项目中计算价值区时使用的成交分布近似方式。

| 模式 | 含义 |
| --- | --- |
| `close` | 将每根 K 的成交量记到收盘价附近 |
| `range` | 将每根 K 的成交量按高低区间分布 |

`close` 更简单，噪声少；`range` 更接近区间成交分布，但可能引入更多近似假设。

## 3. 接受 / 拒绝行为词

### Acceptance / 接受

价格离开原区间后，能够在新区间维持成交。

例子：

```text
价格突破 VAH 后持续在 VAH 上方成交
→ 市场可能接受更高价值。
```

### Rejection / 拒绝

价格尝试进入新区间，但很快回到原区间。

例子：

```text
价格突破 VAH 后很快跌回 VA 内
→ 市场暂时拒绝更高价值。
```

### Reacceptance / 重新接受

价格短暂离开价值区后，重新回到原价值区内。

项目中的典型入口：

```text
跌破 VAL → 收回 VAL 上方 → 旧价值区重新接受，尝试做多回 POC
上破 VAH → 收回 VAH 下方 → 旧价值区重新接受，尝试做空回 POC
```

### Failed Breakout / 假突破

价格突破某个边界后没有延续，反而回到边界内。

项目中常与 `reacceptance` 近似使用。

### Failed Auction

Market Profile / Auction Market Theory 语言，表示市场尝试拍卖到更高或更低价格失败。

项目中可理解为：

```text
价格尝试离开当前价值区，但新区间没有被接受。
```

## 4. 项目结构塑形术语

### 共识价格区间

市场参与者可能共同关注、交易、止损或加仓的价格区域。

候选包括：

```text
前日高低点
开盘区间边界
Initial Balance 高低点
VAH / VAL / POC
密集成交区边缘
支撑阻力密集区
```

### 严格失败边界 / Strict Failure Boundary

如果价格触达或收盘越过该位置，说明本次方向假设理论上失败。

项目中强调：

```text
严格失败边界不是普通止损优化参数，
而是交易假设被证明错误的位置。
```

### Strict Failure Close

价格收盘越过严格失败边界后退出。

比盘中触碰更宽松，但仍表示方向假设失败。

### 实际止损边界 / Stop Price

真实执行止损的位置。它可以等于严格失败边界，也可以在诊断中适度放宽。

### 止损放宽 / Stop Widen

将实际止损放到严格失败边界之外，用来吸收正常噪声。

项目约束：

```text
止损放宽必须同步降低仓位，
保证单次账户风险仍在 2%~3% 预算内。
```

### 短期盈利上界

本次交易在有限持仓周期内可以预先估计的目标空间。

在价值区研究中，常见候选是：

```text
POC
VA 中点
对侧 VA 边界
固定 R 目标
```

### 价格原始盈亏比 / price_raw_rr

入场前按价格距离计算的原始盈亏比。

公式：

```text
price_raw_rr = 入场到盈利上界距离 / 入场到严格失败边界距离
```

项目中用于判断：

```text
这笔交易是否有足够价格空间值得做。
```

### 账户原始盈亏比

按账户风险预算换算后的原始盈亏比。

项目中要求：

```text
失败时账户损失应能控制在 2%~3%，
盈利上界对应的账户收益应显著大于成本。
```

## 5. 本轮价值区研究中的核心自造参数

### min_target_ticks

入场价到目标价之间至少需要多少个 tick 的距离。

在价值区回归研究中通常表示：

```text
入场到 POC 至少要有多少最小价格跳动单位。
```

通俗理解：

```text
离目标太近没肉吃，不做。
```

### min_price_raw_rr

最低价格原始盈亏比要求。

例如：

```text
min_price_raw_rr = 0.5
```

表示：

```text
盈利空间至少要达到严格失败距离的一半。
```

### min_reaccept_ticks

重新回到价值区后，收盘价距离边界至少要有多少 tick。

用途：过滤只“刚好蹭回边界”的弱重新接受。

### max_breakout_bars

价格离开价值区边界后，最多允许在边界外停留多少根 bar。

用途：过滤在边界外停留太久后才回来的弱假突破。

### path_check_bars

入场后检查路径质量的 bar 数。

项目曾测试：

```text
入场后 N 根 5m bar 内，价格是否至少向目标推进一点。
```

结论：简单路径早退不具备统一效果，尤其会伤害慢回归品种。

### min_path_progress_ticks

路径检查期内，价格至少向目标推进多少 tick。

配合 `path_check_bars` 使用。

## 6. 交易退出原因 / Exit Reason

### take_profit

达到目标价主动止盈。

价值区研究中通常表示：

```text
价格回到 POC 或预设目标。
```

### stop_loss

触发实际止损价退出。

### strict_failure_close

收盘价越过严格失败边界退出。

### time_exit

持仓超过最大 bar 数仍未到目标，也未触发失败，按时间退出。

通俗理解：

```text
给它一段时间，不动就走。
```

### force_flat

到强制平仓时间退出，通常用于避免接近收盘或不可控时段持仓。

### path_failure

路径质量检查失败退出。

当前研究结论：该退出方式不能作为统一规则。

## 7. 绩效与路径诊断词

### MAE / Maximum Adverse Excursion

入场后，交易过程中出现过的最大不利浮动。

通俗理解：

```text
这笔交易过程中最多曾经亏到多少。
```

### MFE / Maximum Favorable Excursion

入场后，交易过程中出现过的最大有利浮动。

通俗理解：

```text
这笔交易过程中最多曾经赚到多少。
```

### entry_bar_range_ratio

入场 K 线波动幅度与严格失败距离的比例。

公式：

```text
entry_bar_range_ratio = 入场 bar 高低差 / 入场到严格失败边界距离
```

用途：诊断进场时是否已经处于较剧烈波动状态。

## 8. 价值区上下文标签

这些标签主要写入交易 `reason` 字段，用于后续 SQL 分桶统计。

### ol / open location

今日开盘价相对前日价值区的位置。

| 值 | 含义 |
| --- | --- |
| `inside` | 开在前日 VA 内 |
| `above` | 开在前日 VA 上方 |
| `below` | 开在前日 VA 下方 |

### cl / close location

前日收盘价相对前日价值区的位置。

取值同 `ol`。

### op / open to POC distance

今日开盘价到前日 POC 的距离桶。

### cp / previous close to POC distance

前日收盘价到前日 POC 的距离桶。

### ocr / open-close POC relation

前日收盘与今日开盘相对 POC 的关系。

用于观察：

```text
开盘是否跨过 POC；
开盘和前收是否位于 POC 同侧。
```

## 9. 日级持续价值区标签

### pd / persistent days

日级价值区连续重叠天数。

| 值 | 含义 |
| --- | --- |
| `1d` | 仅确认单日前日价值区 |
| `2d` | 连续 2 天价值区明显重叠 |
| `3d_plus` | 连续 3 天及以上价值区明显重叠 |

### ov / overlap bucket

相邻价值区重叠程度分桶。

| 值 | 含义 |
| --- | --- |
| `low` | 重叠低 |
| `mid` | 中等重叠 |
| `high` | 高重叠 |

### ps / POC stability

日级 POC 稳定性。

| 值 | 含义 |
| --- | --- |
| `stable` | POC 漂移较小 |
| `mild_drift` | POC 温和漂移 |
| `drift` | POC 明显漂移 |

## 10. 日内上下文与滚动价值区标签

### ctx / context

当前使用的上下文窗口。

可能取值：

| 值 | 含义 |
| --- | --- |
| `1h` | 单根 1h 高周期 K 上下文 |
| `2h` | 单根 2h 高周期 K 上下文 |
| `4h` | 单根 4h 高周期 K 上下文 |
| `roll12` | 最近 12 根 5m，约 1h 滚动窗口 |
| `roll24` | 最近 24 根 5m，约 2h 滚动窗口 |
| `roll48` | 最近 48 根 5m，约 4h 滚动窗口 |

当前结论：

```text
roll12 / roll24 / roll48 比 1h / 2h / 4h 单根 K 更适合作为日内价值区诊断。
```

### ctxloc / context location

入场价相对上下文价值区的位置。

| 值 | 含义 |
| --- | --- |
| `inside` | 入场价在上下文 VA 内 |
| `above` | 入场价在上下文 VA 上方 |
| `below` | 入场价在上下文 VA 下方 |
| `na` | 上下文不可用 |

### ctd / context target distance

入场价到上下文 POC 的方向有效距离分桶。

注意：当前研究中，滚动 POC 多数离入场较近，因此更适合作为状态变量，不适合作为直接止盈目标。

### crr / context raw RR

用上下文 POC 作为目标时计算出的价格原始盈亏比分桶。

### cpb / context persistence bars

上下文价值区持续性标签。

在滚动窗口中：

| 值 | 含义 |
| --- | --- |
| `1b` | 当前滚动 VA 与前一段同长度滚动 VA 重叠不足 |
| `2b` | 当前滚动 VA 与前一段同长度滚动 VA 明显重叠 |

### cov / context overlap

上下文价值区重叠程度。

取值同 `ov`：`low` / `mid` / `high`。

### cps / context POC stability

上下文 POC 稳定性。

取值同 `ps`：`stable` / `mild_drift` / `drift`。

## 11. 距离和比例分桶

### td / target distance

入场价到实际交易目标的距离桶。

在价值区回归研究中，多数表示：

```text
入场到前日 POC 的距离桶。
```

### rr / raw RR bucket

`price_raw_rr` 分桶。

常见桶：

| 值 | 含义 |
| --- | --- |
| `lt0_5` | < 0.5 |
| `0_5_1` | 0.5 ~ 1 |
| `1_1_5` | 1 ~ 1.5 |
| `ge1_5` | >= 1.5 |

### vr / volatility ratio bucket

`entry_bar_range_ratio` 分桶。

用于描述入场 bar 波动相对失败距离是否过大。

## 12. 本轮价值区研究的阶段性解释

### 稳定价值区间做 T

对 r3-r8 价值区研究的通俗概括：

```text
如果市场在某个价格区间内反复成交，
区间边缘是低成本试错位置，
成交核心 POC 是短期止盈位置。

价格短暂离开边缘后重新回到区间，
尝试从边缘做到核心。
```

当前阶段结论：

```text
这条线有局部 alpha 线索，
但跨品种泛化能力不足，
更像特定品种 / 特定市场风格下的状态型 alpha，
暂不作为结构型 alpha 主线继续推进。
```

### POC + 空间 / RR

r3/r4 中最有潜力的结构线索：

```text
POC 不是直接信号；
只有当入场到 POC 有足够空间，
且入场到失败边界的距离可控，
POC 才可能成为有意义的盈利上界。
```

对应参数：

```text
min_target_ticks >= 8
min_price_raw_rr >= 0.5
```

当前定位：

```text
重要前置筛选条件，
但不是单独可上线策略。
```
